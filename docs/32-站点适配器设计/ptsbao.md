## 三、站点完整规则

> 来源：`rules.php`

### 3.1 总则

- 不要做管理员明文禁止的事情。
- 不允许发送垃圾信息。
- 一切作弊的账号会被封。
- 注册多个烧包账号将被禁止。
- 不要把本站种子文件上传到其他 Tracker。
- 第一次捣乱行为受警告，第二次永久封禁。

### 3.2 账号保留规则

| 等级 | 规则 |
|------|------|
| 妃 (Veteran User) 及以上 | 永远保留 |
| 贵人 (Elite User) 及以上封存 | 不删除 |
| 封存账号 | 连续 **400 天**不登录 → 删除 |
| 未封存账号 | 连续 **150 天**不登录 → 删除 |
| 无流量账号 | 连续 **150 天**不登录或注册满 **360 天** → 删除 |
| H&R 不达标 | 也会导致账号删除 |

注意：等级名使用**宫廷风**命名（妃、贵人、贵妃、皇后、客卿、带刀侍卫、太监总管等）。

### 3.3 H&R 规则

1. 系统如开启 H&R，默认 **7 天内做种至少 24 小时**（暂定）。
2. 特定规则在种子详情页提示，仅适用于该种子。
3. 下载超过种子总大小 **80%**（暂定）后开始触发 H&R。
4. 单种分享率超过 **1.1**（暂定）则不计算 H&R。
5. 累计得到 **50 次** H&R → 账号自动禁用。

### 3.4 违规处罚规则

**封禁**：
- 上传政治敏感话题资源
- 被警告 3 次自动封禁

**警告（1 周）**：
- 群聊区刷屏/广告/垃圾/色情/政治敏感
- 攻击挑动他人、多次恶意灌水
- 发布不规范种子被删后不改仍发布
- 不尊重发布者
- 标题/正文脏话、纯表情等无意义评论
- 恶意上传不合格字幕或恶意举报
- 发布资源与标题简介严重不符

**警告（2 周）**：
- 任何形式向管理员挑衅、辱骂管理员
- 讨论禁忌/政治敏感话题
- 攻击挑动他人情节严重

### 3.5 上传总则

- 上传者必须对上传的文件拥有合法的传播权。
- 新发布资源 **48 小时内未做种**被举报 → 警告一周，恶意不改 → 限制权限。
- 做种时间不足 **48 小时**或故意低速上传 → 警告甚至取消上传权限。
- **发布者获得双倍上传量。**
- 请勿夹带种子文件或广告（尤其是水印严重的视频）。

### 3.6 禁止的资源

- 政治敏感话题资源
- xv/qsv 等需要专属播放器的视频文件（所有版块禁止）
- 同一资源已有高清版本且非死种时，不允许发布低质量版本（如 RMVB）
- 电影/蓝光/剧集/MV/动漫/纪录片版块禁止 flv/f4v 及 RAR 压缩文件
- **动漫版块禁止发布猪猪字幕组资源**
- 除软件/PC 游戏/掌机/其他版块外，其他版块禁止 RAR 压缩包
- 标清 upscale 视频
- 分卷压缩的游戏或资源
- 劣质标清（CAM 等）
- 无来源和编码信息的影片
- RealVideo（RMVB/RM）、FLV（游戏视频等特殊文件除外）
- 单独的样片
- 无正确 CUE 表单的多轨音频文件
- 禁忌/敏感内容
- 损坏文件、垃圾文件

### 3.7 资源打包规则

**允许打包**：
- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 同一艺术家 MV（标清按 DVD 打包，高清分辨率须相同）
- 同一艺术家音乐（两年内专辑可单独发布）
- 分卷发售动漫剧集/角色歌/广播剧
- 发布组打包资源

**要求**：视频须来源媒介相同、分辨率相同、编码一致；电影合集发布组须统一。音频须编码格式一致。

### 3.8 标题命名规则

**总则**：
- 不使用全角符号标点和数字（如【】〖〗『』１２３）
- 不出现广告等无关信息
- 不加入个人 ID
- 不出现吸引眼球的文字/符号/噱头/个人评价（如"超清晰"、"经典"、"不看后悔"等）
- 不含"求种"、"求置顶"、"推荐"等词汇
- 不出现不必要的空格
- 名称应能让人了解种子内容

### 3.9 促销规则

**随机促销**：

| 概率 | 促销类型 |
|------|----------|
| 8% | 50% |
| 5% | Free |
| 9% | 2x |
| 1% | 2xFree |
| 7% | 2x50% |
| 6% | 30% |

**自动促销**：文件总体积 > **200GB** → 自动 Free。

### 3.10 置顶规则

按分类有不同置顶标准：
- **电影**：最新电影 mkv ≥DVDRip、IMDb≥7.0、优秀合集、经典难获取、做种<3 的优秀资源
- **剧集**：新剧第一集/热播新剧 ≥720P 置顶 1 天；热播合集 ≥720P 置顶 5 天
- **综艺**：下载量高/周更新/优秀晚会典礼/高清资源
- **动漫**：新番第一集置顶 1 天，合集置顶 2 天
- **音乐/MV**：最新专辑置顶 3 天
- **学习**：学习类资源原则上均做促销
- **纪录片**：期待度高/评分高/内容经典优先
- **软件/游戏**：一般不考虑置顶

## 四、适配器设计注意事项

### 4.1 字段映射特点

| 特点 | 说明 |
|------|------|
| mode=1 后缀 | 字段名带 `[1]` 后缀 |
| source_sel 80 选项 | 来源极多，需精确匹配分类×子类型组合 |
| processing_sel 存在 | 区分 Remux/重编码/源码/原盘/AI |
| AI 双重体现 | medium_sel 有 AI(11)、processing_sel 有 AI(6) |
| H.265 值=6 | 非标准编号，H.264=1 但 H.265=6 |
| 杜比视界标签 | tags 有独立杜比视界(8)选项 |
| 宫廷风等级 | 用户等级名使用宫廷风命名 |
| 猪猪字幕组禁止 | 动漫版块禁止猪猪字幕组资源 |

### 4.2 全局禁止资源

- **9KG/色情/成人内容**：source_sel 中含成人-有码(81)、成人-无码(80)、H漫画(84)、H动漫(85)——全局禁止发布
- **政治敏感话题**
- **猪猪字幕组**动漫资源

### 4.3 source_sel 映射策略

source_sel 80 个选项按"分类-子类型"二级结构组织。适配器需要：
1. 根据源站分类确定一级分类（电影/剧集/综艺等）
2. 根据源站属性确定二级子类型（如大陆/美剧/日剧）
3. 组合查找对应的 source_sel 值
4. 无法精确匹配时 fallback 到该分类的"其他"选项
