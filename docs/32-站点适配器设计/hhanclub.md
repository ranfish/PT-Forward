## 三、发种规则（rules.php）

### 3.1 上传总则

- 上传者必须对上传的文件拥有合法的传播权
- 做种时间不足24小时或故意低速上传将被警告甚至取消上传权限
- **非本站小组成员禁止使用本站官组名称作为标识和后缀**
- 发布一个新种子得到50憨豆奖励，上传字幕得20憨豆
- 发布者获得该种子正常实际大小的上传量（非双倍）

### 3.2 上传者资格

- 任何人都能发布资源
- 发布信息多次填写错误/不规范可能被警告、强制候选甚至取消上传权限

### 3.3 允许的资源

- 高清视频（Blu-ray/Remux/重编码≥720p）
- 标清视频（来源于高清的≥480p重编码）
- 7日内高清预告片
- 高清相关软件和文档
- 体育标清视频（TV/DSR）
- <100MB 高清软件/文档/单曲专辑

### 3.4 不允许的资源

- 标清 upscale 视频
- 被管理员判定的有素材制作疑问的资源

### 3.5 Dupe 规则

- **从他站转载官组 WEB 资源时，搜索本站官组是否已发布**
- 同质量、同分辨率的 WEB 资源会被判定为 Dupe
- **官组发布后的资源会把之前同质量、同分辨率资源删除**（但保留不同作品）
- **禁止转载 HDR+DV 种子**，请单独发布 HDR 或 DV 版本

### 3.6 打包规则

- 高清电影合集（套装）
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 打包视频须：相同媒介、相同分辨率、相同编码
- **禁止发布无意义非发行方打包 DIY/Remux 合集**
- **禁止一个 IMDB/豆瓣号下正片多分辨率打包**（除非审核组允许）

### 3.7 保种组规则

- 保种组成员不得对种子限速（临时≥100KB/s，长期≥1MB/s）
- 收到管理组 PM 需24小时内回应
- 累计3次未回应将被降级

### 3.8 种子促销

- 新种子默认 free 8h（不区分是否官种）
- 大体积种子可用憨豆购买促销：免费1天(7,000)/置顶1天(14,000)/置顶+免费1天(27,000)

## 四、HHanClub 特殊注意事项

### 4.1 全站官种

站点全站官种，**只能做源站，不能做发布站**。在 PT-Forward 中应标记为 `PublishEnabled=false`、`SourceEnabled=true`。

### 4.2 Checkbox 筛选系统

种子列表使用 checkbox 多选筛选（非下拉框），字段名带 `[]` 后缀。这使得 RSS URL 构造和页面解析与标准 NexusPHP 不同。

### 4.3 processing 地区分类

`processing[]` 提供剧集地区分类（美剧/日剧/韩剧/港剧/台剧/大陆剧/英剧），是已采集站点中地区分类最细的之一。

### 4.4 独立封面/截图上传

发布页面有 `cover` 和 `screenshot` 独立文件上传字段，标准 NexusPHP 无此字段。

### 4.5 禁止 HDR+DV 合并发种

规则明确禁止转载 HDR+DV 种子，需单独发布 HDR 或 DV 版本。源站消费时需注意拆分。

### 4.6 TAA 音频编码

`audiocodec[]` 含 TAA(18) 和 ALAC(19)，较罕见。

### 4.7 憨豆货币

站点使用"憨豆"货币系统，非魔力值。

## 六、适配器实现要点

### 6.1 仅源站模式

```go
adapter.PublishEnabled = false
adapter.SourceEnabled = true
adapter.OfficialOnly = true
```

### 6.2 Checkbox 筛选 URL 构造

RSS/列表请求需使用 checkbox 参数格式：

```
torrents.php?cat[]=401&medium[]=10&standard[]=1
```

### 6.3 地区信息提取

```go
func mapProcessing(val int) string {
    switch val {
    case 3: return "US"
    case 4: return "JP"
    case 5: return "KR"
    case 6: return "HK"
    case 7: return "TW"
    case 8: return "CN"
    case 9: return "GB"
    default: return ""
    }
}
```
