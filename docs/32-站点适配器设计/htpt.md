# 海棠 站点适配器设计

> **⚠️ 暂不做软件支持** — 用户明确指示仅收集数据，不适合做源站和发布站。

## 站点信息

| 项目 | 内容 |
|------|------|
| 站点名称 | 海棠（HTPT） |
| 站点地址 | https://www.htpt.cc |
| 站点框架 | NexusPHP |
| Tracker URL | `https://www.htpt.cc/announce.php` |
| 站点定位 | **戏曲类特色站**（相声/评书/戏曲/鼓琴/小曲/小品/二人转/小剧种） |
| 规则页面 | rules.php |
| 用户等级名 | 戏曲主题（文丑/Veteran、武生/Elite、票友/Power User 等） |
| 官方发布组 | **HTPT**（官组资源须在标题和种子后缀加 `-HTPT`） |
| **转发定位** | **不适合做源站和发布站** |

---

## 不适合做源站和发布站的原因

1. **戏曲类特色站**：内容全部为戏曲/曲艺相关，与常规影视资源完全不同
2. **官组禁转**：禁止将带有 `-HTPT` 后缀的资源转载至他站，一经发现封号处理
3. **格式要求特殊**：需转成 MP4 或 MP3 格式（考虑唱戏机等特殊播放设备）
4. **禁止转载 VIP 付费内容到本站**
5. 内容范围极窄，常规影视资源无法转发到本站

本站仅做规则记录，不实现适配器。

## 一、发布页面表单字段分析

**提交地址**: `takeupload.php`（POST multipart/form-data）

**注意**: 字段语义完全重定义——戏曲特色，非标准影视站字段体系。

### 1.1 基础信息字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| `file` | file | ✓ | 种子文件 |
| `name` | text | - | 标题（须按标准命名） |
| `small_descr` | text | - | 副标题（如：主演：XXX \| 分辨率：1920*1080 \| 时长：01:55:32） |
| `url` | text | - | 外部信息链接 |
| `nfo` | file | - | NFO 文件（俘虏/Peasant 以下不可见） |
| `descr` | textarea | ✓ | 简介（BBCode） |
| `uplver` | checkbox | - | 匿名发布 |

**缺失**：无 `pt_gen`、无 `durl`（豆瓣）、无 `technical_info`（MediaInfo）、无 `season`/`episode`、无 `screenshot_input`

### 1.2 类型字段（`type`）— 8 个戏曲分类

| 值 | 显示名称 |
|----|----------|
| 1 | 相声 |
| 4091 | 评书 |
| 4097 | 戏曲 |
| 4098 | 鼓/琴 |
| 4099 | 小曲 |
| 4101 | 小品 |
| 4104 | 二人转 |
| 4105 | 小剧种 |

**注意**：分类 ID 极不连续（1, 4091, 4097-4105）。全部为戏曲/曲艺分类，无电影/电视剧/动漫等常规分类。

### 1.3 编码（`codec_sel`）— 3 个

| 值 | 显示名称 |
|----|----------|
| 1 | H.264/AVC/x264 |
| 10 | H.265/HEVC/X.265 |
| 11 | MP3/音频/M4A |

**注意**：值不连续。**MP3/音频/M4A 被视为一种"编码"**，因为站点有大量音频资源。编码字段兼管音视频格式。

### 1.4 分辨率（`standard_sel`）— 4 个（音频勿选）

| 值 | 显示名称 |
|----|----------|
| 1 | 4K |
| 2 | 1080p |
| 3 | 720p |
| 5 | SD/标清 |

**注意**：值不连续（无 4）。标签明确标注"音频勿选"。音频资源只选编码（MP3/音频/M4A），不选分辨率。

### 1.5 制作组（`team_sel`）— 1 个

| 值 | 显示名称 |
|----|----------|
| 1 | HTPT |

**注意**：仅有一个官组 HTPT。用户自行压制的资源不选制作组。

### 1.6 标签（`torrent_tag[]`）— 7 个

| 值 | 显示名称 |
|----|----------|
| jz | 禁转 |
| gf | 官方 |
| yq | 应求 |
| gq | 高清音质 |
| mp3 | MP3格式 |
| mp4 | MP4格式 |
| gc | 国粹之声 |

**注意**：
- 标签值使用**中文拼音缩写**（jz=禁转、gf=官方、yq=应求等），非数字 ID
- 有 MP3 格式(5) 和 MP4 格式(6) 标签——对应唱戏机播放需求
- 字段名为 `torrent_tag[]`（非 `tags[]` 或 `tag_list[]`）

### 1.7 缺失字段

- 无 `medium_sel`（媒介）
- 无 `audiocodec_sel`（音频编码）—— 合并到 `codec_sel`
- 无 `processing_sel`（处理/地区）
- 无 `source_sel`（来源）

## 二、字段语义对照表

| 字段名 | 常见站点含义 | 海棠含义 |
|--------|-------------|---------|
| `type` | 电影/电视剧/动漫等 | **戏曲分类**（相声/评书/戏曲/鼓琴/小曲/小品/二人转/小剧种） |
| `codec_sel` | 视频编码 | **音视频编码合一**（H.264/H.265/**MP3/音频/M4A**） |
| `standard_sel` | 分辨率 | 分辨率（**音频勿选**） |
| `team_sel` | 制作组（多个） | **仅官组 HTPT**（1 个） |
| `torrent_tag[]` | 数字 ID | **拼音缩写**（jz/gf/yq/gq/mp3/mp4/gc） |

## 三、发种规则（rules.php）

### 3.1 发布者资格

- 普通票友需先在候选区提交候选
- 发布资源类型必须与海棠 PT 的戏曲特色相符合
- **禁止发布任何站点的"付费""点播""禁转"等容易引起争论的资源**
- **HTPT 官方授权发布员须在标题和种子后缀加 `-HTPT` 管组字样**

### 3.2 资源质量要求

- 自行压制/编辑/剪切/转载的资源须先检查质量
- **视频模糊、音质不清晰的直接弃掉**
- 资源内带广告和时长不完整的直接弃掉（戏曲精选唱段除外）
- **视频资源 LOGO 超过两个不得发布**
- 文件名中杂质信息须删除

### 3.3 格式要求

- **发布资源应转成 MP4 或 MP3 格式**（考虑唱戏机等特殊播放设备）

### 3.4 标题命名规范

**普通资源**：`【主演】- 剧种 -《剧名》全剧-HTPT`
- 例：`【赵志刚 方亚芬】- 越剧 -《红楼梦》全剧-HTPT`

**副标题**：`[禁转][MP4格式] | 分辨率：1920*1080 | 时长：02:52:11`
- `[禁转][MP4格式]` 为彩色标签

**电影资源**：`英文名 年份 来源 分辨率 音频编码-HTPT`
- 例：`Fighting with Python Monster 1961 WEB-DL 720P AAC-HTPT`

### 3.5 简介要求

- 戏曲电影资源，豆瓣有封面的务必贴上封面
- 尽量贴上曲目的内容简介（相声和"专辑包"资源除外）
- 每个信息区块以 `[quote]` 开始、`[/quote]` 结束
- 须提供 MediaInfo 信息

### 3.6 做种要求

- 发布资源后不得限速小于 **1MB/秒**
- 未出种前不得删除资源

## 四、促销规则

### 4.1 新发布资源

- **100% 概率免费**（新发布资源自动免费）
- 免费限时 **7 天**
- 7 天结束后，**永久只计算 50% 下载量**

### 4.2 发布者奖励

- 发布者获得**双倍上传量**

## 五、账号保留规则

| 条件 | 处理 |
|------|------|
| Veteran User（文丑）及以上 | 永远保留 |
| Elite User（武生）及以上封存账号 | 不会被删除 |
| 封存账号连续 365 天不登录 | 删除账号 |
| 未封存账号连续 180 天不登录 | 删除账号 |
| 无流量用户连续 90 天不登录 | 删除账号 |

**注意**：等级名称使用戏曲主题（文丑=Veteran、武生=Elite、票友=Power User 等）。

## 六、转载规则

- 将本站资源转载至他站时，务必注明"转载自海棠 PT"
- **禁止将带有 `-HTPT` 后缀的资源转载至他站**，一经发现封号处理
- **严禁转载 VIP 付费内容到本站**
