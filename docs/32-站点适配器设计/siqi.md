## 站点特色

**图像书特色站**，PT 圈中极为罕见的图书专题站点：

- 资源必须是**图像书**（书籍内图像需占全书面积 1/3 以上）
- 优先格式：CBZ/CBR、EPUB、PDF、图片格式
- 低质量 MOBI 可能不予通过
- 禁止发布纯粹卖肉的"9kg"内容（艺术向刊物、写真集等除外）
- 资源包大小建议 100MB - 10GB，**禁止超过 100GB 的大包**
- 提供**自动发种程序**辅助工具
- 有**自购自扫**资源类别（需提供证明，有额外魔力奖励）

## 发种规范（来源：forums.php?forumid=1&topicid=1）

### 标题格式
```
书名（原书名或其他语言名）（补充说明）[格式说明]
```
- 翻译本：附上原书名
- 多种格式：在中括号加格式说明如 `[PDF-EPUB-MOBI]`
- 自购自扫资源必须尽量完善标题

### 副标题格式
```
作者名｜出版年份｜出版社｜ISBN
```
- 多作者用中文分号；分隔
- 译者加：译者：xxx, xxx
- 合集类先写文件格式，用｜分隔关键词

### 简介要求

**必须包含的声明（自动添加）：**
> 本站所有资源均是用户在网络上收集且上传发布的，本站不存储任何资源文件，且本站对用户所发布的所有资源内容一无所知。本站内容仅做宽带网速测试、学习交流等使用，请您在下载后24小时内尽快删除。转载本站官方资源至他站时，请务必注明此资源转载自思齐PT。

**转载额外说明：**
> 本资源转载自xxx，感谢原作者发布。

**自购自扫额外说明：**
> 本资源为自购，请勿转到其他站点以及任何的公共平台，违者一律封禁账号。

**单本简介：**
1. 封面图
2. 豆瓣内容：图书元数据（含豆瓣链接）、内容简介、作者介绍
3. 至少 3 张图书内页截图（建议选图像内容突出部分）
4. 自购自扫需增加自购自扫证明图

**合集简介：**
1. 简要介绍（建议使用 AI 辅助生成）
2. 至少 3 张代表性截图
3. 自购自扫严禁合集，必须单本（或漫画合集）

## 质量字段

**无任何标准 NexusPHP 质量下拉框。** 没有 medium_sel、codec_sel、audiocodec_sel、team_sel、resolution_sel、source_sel、processing_sel。这与站点只发布图像书资源的特性一致。

upload.php 表单未能获取（可能需要 Power User 权限才能看到完整表单）。

## 豆瓣集成

- 发种参考要求使用**豆瓣搜书**获取图书元数据
- 简介需包含豆瓣链接
- 豆瓣内容（元数据、内容简介、作者介绍）为必填项

## 违规处理

1. 预览图必须来自种子内资源，上传无关图片首次警告+禁下载一个月
2. 连续发布大量不合规种子将被警告，再次违规可能禁号
3. 发种被拒后 24 小时内不回应或修改将被清除
4. 禁转标签的自购资源不得转到其他站点及公共平台，违者 BAN
5. 严禁拆分本站已有大包重新发布

## 自购自扫资源特殊规则

1. 需提供自购自扫证明（扫描仪扫书照片）
2. 只接受单本（或漫画合集）
3. 需完善元数据填写，格式为 PDF 或压缩图片格式
4. 只接受公网或其它 PT 站无法找到的独家资源
5. 扫描质量：分辨率不低于 300ppi
6. 分两步审核：初审（元数据）+ 终审（独家确认）
7. 通过终审获随机盲盒奖励（5000-50000 魔力）

## 特殊说明

1. **纯图像书站**：PT 圈罕见图书专题站，不适合做源站和发布站
2. **无类似站点**：没有其它 PT 站与思齐类似，无法从其它站转发图书资源
3. **无质量字段**：图书资源不涉及视频编码/音频编码/分辨率等传统质量维度
4. **豆瓣集成**：深度集成豆瓣图书数据，简介必须包含豆瓣链接和内容
5. **自动发种程序**：站点提供自动发种工具辅助发布
6. **Power User 发种**：需要 Power User 等级才能发布
7. **预览图严格要求**：1 封面 + 至少 3 内页截图，必须来自种子内资源
8. **杂志合集规则**：连续年份完整的同一杂志需发合集，只有最新的可发单本
