## 站点特色

- **官组非常活跃**：站组 UBits（原盘DIY组）、UBWEB（流媒体组）、UBTV（电视录制组），官种量大
- **仅源站角色**：官组活跃+种审制，不适合做发布站
- **种审制**：发布后需审核通过才能正式显示，候选积分制管理
- **编码含AVS**：codec_sel包含国产AVS编码
- **音频编码细分**：Atmos/DTS:X/TrueHD/DTS-HD MA/HR/DD+/DD独立选项，共14种
- **分辨率含1440p**：standard_sel包含1440p
- **地区10个**：含中国大陆/港/台/欧美/日/韩/泰/印/俄/其它
- **双语规则**：论坛发种规则中英双语
- **禁止删除原站标识**：转载时保留原站标识（如DIY@UBits），删除者删种+警告+取消发布权限
- **与不可说互斥**：禁止互相转载发布
- **黑名单站点/制作组**：HDSky（-HDSky/@HDSky/-HDS/-HDSWEB/-HDSTV）、CMCT（-CMCT/@CMCT/-CMCTV/-CMCTA）资源禁止发布
- **禁止官种转载至黑名单站**：PTTime、HDSky、CMCT（SpringSunDay）
- **发布者双倍上传量**（仅官组种子）

## ⚠ 转载互斥规则

## 标题命名规范（topicid=269/1620）

### 电影
- 英文名称 + 年份 + 分辨率 + 来源 + 视频编码 + 音频 + 发布组名称
- 例：`The Invisible Guest 2023 1080p CATCHPLAY WEB-DL H264 AAC-UBWEB`

### 电视剧
- 英文名称 + 年份 + 季/分集 + 分辨率 + 来源 + 视频编码 + 音频 + 发布组名称
- 例：`The Queen of Castle 2025 S01E23 1080p WEB-DL H264 AAC-UBWEB`

### 通用规则
- 主标题**不能含中文**
- 保留原发布站标识（如"DIY@UBits"、"UBits"）
- 删除原标识者：删种+警告一周+取消发布权限一周
- 各元素顺序不要求一致，但各元素都要含有

### 副标题
- 影片中文名称，如"特别响，非常近 / 心灵钥匙(台) / 咫尺浩劫(港)"
- 可附加"蓝光原盘"、"4K UHD"等信息

### 简介要求
- 必须包含：**豆瓣海报**、简介文字、资源参数（BDinfo/Mediainfo）
- MediaInfo/BDInfo 必须填写在 `technical_info` 字段
- 转载自其他站点，须在简介上方用 `[quote]` 标签标注来源
- IMDB 和豆瓣：如简介中有豆瓣（国外剧）或 IMDB（国产剧/综艺），可只填一个

## 种审制（详细规则 topicid=1548）

### 候选积分制
- 候选积分 < 10 的用户发布的资源 → 自动进候选区
- 体积超过 100GB 的大型资源 → 自动进候选区
- 与官方资源重复的发布 → 自动进候选区
- 被管理组标记为发布不规范的用户 → 候选积分为负值时

### 候选积分计算
| 事件 | 积分变化 |
|------|---------|
| 种子审核从"待定"→"通过" | **+1** |
| 种子审核从"未审"→"拒绝" 或 "通过"→"拒绝" | **-1** |
| 从"待定"→"拒绝" 或 "拒绝"→"通过" | 不处理 |
| 积分下限 | 不会自动降至负值，但管理组可手动设为负值 |

### 注意事项
- 发布后**必须立即做种**，未做种不予审核
- 待审核状态**不影响**其他用户连接做种
- 系统定时清理长时间未通过的候选种子

### 种审要点（topicid=1620）
1. 种子标题命名是否规范
2. 标签是否正确（原生原盘/DIY/国语/粤语/中字/HDR系列等）
3. 基本信息：媒介/视频编码/音频编码/分辨率/地区
4. 简介：海报、资源信息、截图（可无）
5. 媒体参数：原盘需 BDInfo，其他影视需完整 MediaInfo
6. IMDB 和豆瓣链接
7. 种子文件：禁止转发 ISO 封装的非 3D DIY/REMUX

## 发布规则

### 禁止内容
- 总体积小于100MB的资源（100M以下不计任何魔力）
- 标清upscale视频
- CAM/TC/TS/SCR等低质量视频
- RMVB/RM/flv文件
- RAR等压缩文件
- 色情/敏感政治内容
- **黑名单站点资源**：HDSky、CMCT 全系列
- **非Untouched抓取的2D原盘以ISO封装**（3D除外）
- **破解软件/游戏破解版/工具软件**（因无法鉴别安全性）
- **人为将合集拆分成小种子**
- **禁止拆包刷流**（下载部分内容只上传，管理员判定，查实封禁）
- **禁止大量跳车刷流**（过于激进的自动删种策略，管理员判定，查实封禁）

### Dupe规则
- 来源媒介优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 动漫类HDTV与DVD同优先级（特例）
- 断种45日或发布18个月以上可重发
- **官方Dupe规则**：官组发布后，除压制外，不接受任何他组同来源同规格版本（流媒体同抓取源且同分辨率/同码率/同大小视为同规格）
- 官方未发布的资源，允许友站资源多版本共存

### 资源打包规则（试行）
- 与通用模板基本一致，额外：**不允许跨季打包，不允许用户自行打包**电视剧
- 允许打包：电影合集/整季剧/专题纪录片/7日内预告片/同艺术家MV/同艺术家音乐(≥5张)/动漫分卷/发布组打包
- 打包约束：相同媒介/分辨率/编码，电影合集须发布组统一

### 账号保留（比通用模板更严格）
- Veteran User及以上：永久保留
- Elite User及以上封存后：不删除
- 封存账号：**90天**未登录禁用
- 未封存账号：**60天**未登录禁用
- 无流量用户：**45天**未登录删除
- 因不活跃被封禁且拥有≥80万做种积分：申诉后解封，扣除80万做种积分
- 自动化工具登录/签到可能不记录为有效访问，最后访问时间以用户详情页面为准

### 字幕区规则
- 允许格式：srt / ssa / ass / cue / zip / rar
- Vobsub格式须打包为 zip/rar
- 禁止上传 lrc 歌词等非字幕/cue 文件
- 不合格字幕直接删除，上传者扣100魔力
- 举报不合格字幕奖励50魔力

### 管理组退休待遇
- 上传员：由当前工作组组长决定（非固定规则）
- 版主：未在规则页详细列出
