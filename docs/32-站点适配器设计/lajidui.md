## 一、总则

### 1.1 账号保留规则

1. Veteran User 及以上等级永远保留
2. Elite User 及以上等级封存账号（控制面板）后不会被删除
3. 封存账号连续 400 天不登录 → 删除
4. 未封存账号连续 150 天不登录 → 删除
5. 零流量用户（上传/下载均为 0）连续 100 天不登录 → 删除

### 1.2 通用规则

- 一切作弊账号被封
- 禁止注册多个账号
- 禁止将本站种子文件上传到其他 Tracker
- 第一次捣乱行为警告，第二次永久封禁

## 二、促销规则

### 2.1 随机促销（种子上传后系统自动随机设定）

| 概率 | 促销类型 |
|------|---------|
| 10% | 50% 下载 |
| 5% | 免费 |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传 |

### 2.2 自动免费

- 文件总体积 **> 20GB** 的种子 → 自动免费
- **Blu-ray Disk / HD DVD 原盘** → 自动免费
- 电视剧等 **每季第一集** → 自动免费
- 关注度高的种子 → 管理员手动设促销

### 2.3 促销时限

- 除"2x 上传"外，其余类型促销限时 **7 天**（自种子发布时起）
- "2x 上传"**无时限**
- 所有种子发布 **1 个月后** → 自动永久成为 **2x 上传**
- 不定期开启全站免费

## 三、上传规则

### 3.1 上传总则

- 上传者必须对上传的文件拥有合法传播权
- 发布者获得 **双倍上传量**
- 做种时间不足 24h 或故意低速上传 → 警告甚至取消上传权限
- 违规种子不经提醒直接删除

### 3.2 上传者资格

- 任何人都能发布资源
- **游戏类资源**：仅上传员及以上等级或管理组特别指定用户可自由上传，其他用户须先在候选区提交候选

### 3.3 允许的资源

- 高清视频：蓝光原碟/HD DVD/remux/HDTV/重编码（≥720p）/高清 DV
- 标清视频：高清媒介的标清重编码（≥480p）/DVDR/DVDISO/DVDRip/CNDVDRip
- 无损音轨（FLAC/Monkey's Audio 等）+ cue 表单
- 5.1 声道及以上音轨（DTS/DTS-CD 镜像等）+ 评论音轨
- PC 游戏（必须原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 3.4 不允许的资源

- 总体积 < **100MB**（例外：高清软件文档、单曲专辑）
- 标清 upscale 视频
- 低质量标清：CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD
- RealVideo/RMVB/RM/FLV
- 单独样片（须与正片一起上传）
- < 5.1 声道的有损音频（MP3/WMA 等）
- 无正确 cue 的多轨音频
- 硬盘版/高压版游戏、非官方镜像、第三方 mod、小游戏合集、单独破解/补丁
- RAR 等压缩文件
- Dupe 资源
- 禁忌/敏感内容（色情、政治敏感等）
- 损坏文件、垃圾文件（病毒/木马/广告/种子中含种子等）

## 四、Dupe 判定规则

### 4.1 来源媒介优先级

**Blu-ray / HD DVD > HDTV > DVD > TV**

- 高优先级版本使低优先级版本被判定为 Dupe
- 高清版本使标清版本被判定为 Dupe
- **动漫特例**：HDTV 和 DVD 版本优先级相同

### 4.2 同媒介同分辨率重编码

- 按发布组优先级判定（参考 "Scene & Internal, from Group to Quality-Degree"）
- **DVD5 保留**：总会保留一个当前最佳画质来源经重编码而成的 DVD5 大小（≈4.38GB）的版本
- 基于无损截图对比，高质量版本使低质量版本被视为 Dupe

### 4.3 豁免条款

- 不同区域/不同配音/字幕的 Blu-ray/HD DVD 原盘 → **不算 Dupe**
- 每个无损音轨原则上只保留一个版本（分轨 FLAC 优先级最高）
- 新版本修复旧版本错误或有更高质量来源 → 允许发布，旧版本被视为 Dupe
- 旧版本连续断种 **45 日以上** 或发布 **18 个月以上** → 不受 Dupe 约束
- 新版本发布后，旧版本保留至断种

## 五、资源打包规则（试行）

### 5.1 允许打包

- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7 日内高清预告片
- 同一艺术家 MV（标清按 DVD 打包，不允许单曲 MV 单独发布；高清分辨率须一致）
- 同一艺术家音乐（≥5 张专辑方可打包；两年内发售专辑可单独发布；剔除站内已有资源或全部包含）
- 分卷发售动漫剧集/角色歌/广播剧
- 发布组打包发布的资源

### 5.2 打包一致性要求

- 视频资源：相同媒介类型、相同分辨率、相同编码格式（预告片例外，电影合集须发布组统一）
- 音频资源：编码格式一致（如全为分轨 FLAC）
- 打包发布后视情况删除相应单独种子

## 六、例外规则

- 允许 TV/DSR 来源的体育类标清视频
- 允许 < 100MB 的高清相关软件和文档
- 允许 < 100MB 的单曲专辑

- 允许附带字幕/游戏破解与补丁/字体/包装扫描图（须统一打包或统一不打包）

## 七、种子信息命名规范

### 7.1 标题格式

| 资源类型 | 格式 |
|---------|------|
| 电影 | `[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组` |
| 电视剧 | `[中文名] 名称 [年份] S\*\*E\*\* [发布说明] 分辨率 来源 [音频/]视频编码-发布组` |
| 音轨 | `[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组]` |
| 游戏 | `[中文名] 名称 [年份] [版本] [发布说明][-发布组]` |

### 7.2 简介要求

- 电影/电视剧/动漫：必须包含海报/封面，尽量包含截图、文件详情（格式/时长/编码/码率/分辨率/语言/字幕）、演职员/剧情
- 音乐：必须包含专辑封面和曲目列表
- PC 游戏：必须包含海报/封面，尽量包含截图
- 体育节目：不得在文字/截图/文件名中泄露比赛结果
- NFO 图须写入 NFO 文件，不粘贴到简介

## 九、字幕区规则

### 9.1 基本规则

- 允许格式：srt / ssa / ass / cue / zip / rar
- Vobsub（idx+sub）或合集须打包为 zip/rar
- 禁止 lrc 歌词或非字幕/cue 文件

### 9.2 不合格判定

与种子不匹配/不同步/打包错误/含无关文件/编码错误/cue 错误/语种标识错误/标题命名不明确/重复/被举报证实

### 9.3 奖惩

- 举报不合格字幕确认后 → 奖励举报者 **50 魔力值**
- 不合格字幕上传者 → 扣除 **100 魔力值**
- 恶意上传或恶意举报 → 额外扣魔力值甚至警告

## 十一、关键适配器设计要点

### 11.1 processing_sel = 文件格式

`processing_sel` 在通常 NexusPHP 站中表示处理方式，但在垃圾堆中表示 **文件格式/封装格式**（MKV/MP4/EPUB/PDF/RAR/ISO 等）。这是第三个字段语义特殊站（继 PTCafe 的 source_sel=地区、YHPP 的 processing_sel=地区之后）。

适配器需根据目标内容类型选择合适的 processing_sel 值：
- 视频资源 → MKV(10) 或 MP4(11)
- 电子书 → EPUB(1) / PDF(2) / TXT(3) 等
- 光盘镜像 → ISO(16)

### 11.2 source_sel = 地区

`source_sel` 表示地区（大陆/港台/日韩/欧美），非来源媒介。

### 11.3 分类覆盖面广

16 个分类涵盖影视、动漫、音乐、少儿、电子书、短剧、游戏、APP、教育、有声书等，综合性强。

### 11.4 2K 分辨率

有独立的 2K 分辨率选项（standard_sel=5），多数站无此选项。

### 11.5 Cloudflare 防护

站点使用 Cloudflare（需 `cf_clearance` cookie）。

## 十三、站点间字段语义对比

| 字段名 | 标准含义 | Lajidui 含义 | 其他特殊站 |
|--------|---------|-------------|-----------|
| `source_sel` | 来源媒介 | **地区** | PTCafe（地区） |
| `processing_sel` | 处理方式 | **文件格式** | YHPP（地区） |
| `audiocodec_sel` | 音频编码 | 音频编码 | — |
| `medium_sel` | 媒介 | 媒介 | — |

**重要结论**：NexusPHP 站间即使字段名相同，含义也可能完全不同。适配器必须按站点配置映射，不能依赖字段名推断。
