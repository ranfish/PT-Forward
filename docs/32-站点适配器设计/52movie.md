## 站点特色

- **规则与织梦/自然完全同模板**：上传总则/Dupe判定/打包规则/促销规则/标题格式与织梦、自然站几乎逐字一致
- **字幕区**：独立字幕上传系统（srt/ssa/ass/cue/zip/rar）
- **认领系统**：每人最多 1000 种
- **签到系统**：每日签到得 500 魔力值
- **随机促销**：与织梦/自然完全一致（10%→50%下载，5%→免费，5%→2x上传，3%→50%下载&2x上传，1%→免费&2x上传）
- **大体积自动免费**：>20GB 免费，Blu-ray/HD DVD 原盘免费，电视剧每季第一集免费
- **发布者双倍上传**
- **游戏类限制**：仅上传员及以上可自由上传游戏
- **9KG/成人内容禁止**：本软件禁止下载和发布此类内容

## 标题命名规范

### 电影
```
[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称
```
示例：`蝙蝠侠:黑暗骑士 The Dark Knight 2008 PROPER 720p BluRay x264-SiNNERS`

### 电视剧
```
[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称
```
示例：`越狱 Prison Break S04E01 PROPER 720p HDTV x264-CTU`

### 音轨
```
[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]
```

### 游戏
```
[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]
```

## Dupe判定规则

与织梦/自然站**完全一致**：

| 规则 | 说明 |
|------|------|
| 媒介优先级 | Blu-ray/HD DVD > HDTV > DVD > TV |
| 高清优于标清 | 高清版本使标清版本被判定为重复 |
| 动漫特例 | HDTV和DVD版本有相同优先级 |
| 同媒介同分辨率 | 按发布组优先级确定 |
| DVD5保留 | 总会保留一个DVD5大小（4.38GB）的版本 |
| 截图对比 | 基于无损截图对比 |
| 不同区域原盘 | 不同配音/字幕不算重复 |
| 无损音轨 | 原则上只保留一个版本，分轨FLAC优先级最高 |
| 断种替换 | 连续断种45日或发布18个月以上 |
