## 重要声明

## 站点特色

- 资源必须是**图像书**（书籍内图像需占全书面积 1/3 以上）
- 优先格式：CBZ/CBR、EPUB、PDF、图片格式
- 低质量 MOBI 可能不予通过
- 禁止发布纯粹卖肉的"9kg"内容（艺术向刊物、写真集等除外）
- 资源包大小建议 100MB - 10GB，**禁止超过 100GB 的大包**
- 有**自购自扫**资源类别（需提供证明，有额外魔力奖励 5000-50000 魔力）
- 不禁盒子，上传速度限制 90MB/s 以下
- 禁止恶意刷流（反复跳车下载拉上传不做种）
- 提供**自动发种程序**和**预览图提取/上传脚本**
- 有**种子审核系统**（approval_status：未审/通过/拒绝）

## 发种规范（来源：forums.php?forumid=1&topicid=1）

### 标题格式
```
书名（原书名或其他语言名）（补充说明）[格式说明]
```
- 翻译本：附上原书名
- 多种格式：在中括号加格式说明如 `[PDF-EPUB-MOBI]`
- 自购自扫资源必须尽量完善标题

### 副标题格式
```
作者名｜出版年份｜出版社｜ISBN
```
- 多作者用中文分号；分隔
- 译者加：译者：xxx, xxx
- 合集类先写文件格式，用｜分隔关键词

### 简介要求

**必须包含的声明（自动添加）：**
> 本站所有资源均是用户在网络上收集且上传发布的，本站不存储任何资源文件，且本站对用户所发布的所有资源内容一无所知。本站内容仅做宽带网速测试、学习交流等使用，请您在下载后24小时内尽快删除。转载本站官方资源至他站时，请务必注明此资源转载自思齐PT。

**转载额外说明：**
> 本资源转载自xxx，感谢原作者发布。

**自购自扫额外说明：**
> 本资源为自购，请勿转到其他站点以及任何的公共平台，违者一律封禁账号。

**单本简介：**
1. 封面图
2. 豆瓣内容：图书元数据（含豆瓣链接）、内容简介、作者介绍
3. 至少 3 张图书内页截图（建议选图像内容突出部分）
4. 自购自扫需增加自购自扫证明图

**合集简介：**
1. 简要介绍（建议使用 AI 辅助生成）
2. 至少 3 张代表性截图
3. 自购自扫严禁合集，必须单本（或漫画合集）

### 预览图要求
- **必须**：1 张封面 + 至少 3 张内页预览图
- 预览图必须来自种子内资源，上传无关图片属严重违规
- 首次违规：警告 + 禁下载一个月

### 豆瓣集成
- 发种参考要求使用**豆瓣搜书**获取图书元数据
- 简介需包含豆瓣链接
- 豆瓣内容（元数据、内容简介、作者介绍）为必填项

## 站点规则（来源：rules.php）

### 总则

- 不允许发送垃圾信息
- 一切作弊账号会被封
- 禁止注册多个账号
- 禁止把本站种子上传到其他 Tracker
- 第一次捣乱警告，第二次永久封禁
- 禁止注册 admin/Administrator/Moderator/Staff Leader 等管理样字眼，会被删号
- **禁止恶意刷流**：反复跳车下载拉上传不做种，第一次警告并禁下载，第二次删除账号
- **盒子不限**：但上传速度限制 90MB/s 以下，第一次警告，第二次禁下载

### 账号保留规则

| 条件 | 处理 |
|------|------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上封存账号 | 不被删除 |
| 封存账号连续 400 天不登录 | 删除 |
| 未封存账号连续 150 天不登录 | 删除 |
| 无流量用户连续 100 天不登录 | 删除 |

### 上传总则

- 发布者获得 1 倍上传量（非双倍）
- 禁止发布 > 100G 的资源包
- 做种时间不足 24 小时将警告甚至取消上传权限

### 发种强调（首页公告）

1. 发图像书看规则带预览图（链接）
2. 禁大包（> 100G）
3. 避已有资源
4. 连续发不符合要求种子将被警告

## 自购自扫资源特殊规则

1. 需提供自购自扫证明（扫描仪扫书照片）
2. 只接受单本（或漫画合集），严禁合集
3. 需完善元数据填写，格式为 PDF 或压缩图片格式
4. 只接受公网或其它 PT 站无法找到的独家资源
5. 扫描质量：分辨率不低于 300ppi，彩色保持彩色，黑白保持黑白
6. 按书籍原始尺寸扫描，避免统一使用 A4 模式
7. 分两步审核：初审（元数据）+ 终审（独家确认）
8. 通过终审获随机盲盒奖励（5000-50000 魔力）
9. 禁转标签自购资源不得转到其他站点及公共平台，违者 BAN

## 违规处理

1. 预览图无关图片：首次警告 + 禁下载一个月
2. 连续发布大量不合规种子：警告，再次违规可能禁号
3. 发种被拒后 24 小时内不回应或修改：将被清除
4. 禁转标签自购资源转出：BAN
5. 拆分本站已有大包重新发布：严禁
6. 发布前需搜索本站是否已有相同资源

## 缺失字段

- upload.php 表单结构未能获取（普通用户无发布权限）
- 无标准 NexusPHP 质量下拉框（medium_sel/codec_sel 等）

## 特殊说明

1. **纯图像书站**：PT 圈罕见图书专题站，没有其它 PT 站类似
2. **无质量字段**：图书资源不涉及视频编码/音频编码/分辨率等传统质量维度
3. **豆瓣深度集成**：简介必须包含豆瓣链接和图书元数据
4. **制作组 SQB**：官种制作组名为 SQB
5. **发布者仅 1X 上传**：非双倍，与多数 PT 站不同
6. **盒子不限但限速**：90MB/s 上传上限
7. **预览图严格要求**：1 封面 + 至少 3 内页截图，必须来自种子内资源
8. **杂志合集规则**：连续年份完整同一杂志需发合集，只有最新的可发单本
9. **种子审核系统**：转载种子有审核流程
10. **多维度次分类**：来源/题材/语种/画面风格/连载状态/文件格式/标签/制作组/品质
