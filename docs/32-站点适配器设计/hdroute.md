## 站点特色

- **老牌原盘/高清站**：以高品质蓝光原盘、Remux、重编码为主，2013年建站
- **自研框架**：完全不同于NexusPHP，URL结构、表单字段名、页面布局均为自研
- **严格的发布权限**：仅允许3类资源——原盘（任何人可发）、原创压制或WEB（仅限原up发布）、特许发布
- **独特的"最高音轨"字段**：不同于标准站点的"音频编码"，HDR强调最高规格音轨
- **三段式标题**：中文标题 + 一句标题 + 英文标题，各有独立字段
- **音轨标记系统**：通过复选框标记国语音轨、粤语音轨、原生中字、DIY等属性（非下拉菜单）
- **用户等级按分辨率命名**：如1080P、720P等
- **无标签系统**、**无制作组下拉菜单**、**无地区下拉**
- **流量赌场**（Casino）特色功能

### 标题命名规则

**原盘/Remux/重编码/miniBD/WEB-DL：**
- 中文标题：影片中文名（自动加【】）
- 一句标题：随意或留空
- 英文标题：`影片英文名.发行年份.资源载体.视频分辨率.视频编码类型.音轨编码类型-制作小组`

**示例：**
```
中文标题：谍影重重1
一句标题：
英文标题：The.Bourne.Identity.2002.BDRip.a720.x264.AC3-R2HD
```

```
中文标题：变形金刚
一句标题：港版D2特别版.蓝光原盘
英文标题：Transformers.2007.Blu-ray.1080P.H264.TrueHD 5.1-DIY
```

**连续剧集：**
- 中文标题：剧集中文名（自动加【】）
- 一句标题：季数
- 英文标题：`剧集英文名.英文季数.分辨率.剧集来源.编码类型-制作小组`

**示例：**
```
中文标题：南方公园
一句标题：第13季
英文标题：South.Park.S13.720p.HDTV.H264-CTU
```

**无损音乐：**
- 中文标题：专辑名称（自动加【】）
- 一句标题：音乐格式.艺术家 - (整轨/分轨)
- 英文标题：有的话留英文，没有留空

**示例：**
```
中文标题：天堂
一句标题：APE.腾格尔 -(分轨)
英文标题：
```

> **注意**：【】为全角方括号，系统自动添加，无需手动输入。禁止修改制作小组名称。

## 上传规则要点

### 资源限制
- **仅允许3类资源**：原盘（任何人可发）、原创压制/WEB（仅原up）、特许发布
- 分辨率要求A720P及以上（HDR官方MP4除外）
- 禁止低质量文件重编码为"伪高质量"
- 连续剧原创Encode/WEB-DL仅限整季打包，禁止单集
- 禁止发布纯色情/政治/反动内容
- 禁止上传分卷RAR等压缩文件
- 禁止上传MP3格式等非无损音频

### Dupe规则
- 相同版本不同小组的种子算重复
- 不同区域或不同版本的盘不算重复
- 死种>365天可重发（不算Dupe）
- 死种>6个月且有人求种可重发
- 不符合现行规则的旧种不论是否有人求种均不可重发

### H&R规则
- 3天内累计保种至少24小时（不管比例）
- 违者警告，两次警告后禁号
- 直到出现3个种子才能撤种
- 如无任何人下载，72小时后可撤种

### 描述要求
- 必须包含：影片简介（建议PTGen）、封面、NFO（mediainfo/BDinfo）、IMDB（电影类必填）
- 条件允许尽量包含截图和花絮内容清单
- 0day资源6小时内补充资料，标题须注明"0day国内首发，资料稍后补充"

### 原盘特殊规则
- 必须标明来源、出处及BDInfo信息
- DIY原盘/D商出版物必须在标题中注明
- 非原盘原抓ISO须在标题注明DIY或DIYISO
- 禁止在原盘内携带垃圾文件
- 允许BDMV或ISO格式，但禁止假ISO和描述不符的ISO（违者永封）

### 字幕规则
- 使用种子页面内字幕功能
- 仅限傲视网或射手网字幕链接

### Uploader发布要求
- 转载：每月≥8种，每年≥100种
- 自购/DIY：每年≥6种
- 上传期间保持连接
- 8周内至少有一个新发资源

### 转发规则
- 禁止转发有"拒绝转发"图标的种子
- 禁止到其他网站辅种被违规转发的种子
- 违反转载规则、提取HDR/R2HD小组制作的国语音轨单独发布或合成到其他影片的会被禁用账号

## 禁止发布内容

- 纯色情、政治、反动、令人厌恶内容
- 压缩文件（RAR等分卷）
- MP3等非无损音频
- 低质量重编码为"伪高质量"
- 假ISO/描述不符ISO

## 源站价值分析

### 作为源站的优势
1. **高品质原盘资源丰富**：专注原盘/Remux，资源品质有保障
2. **严格的质量控制**：Uploader制度+严格审核
3. **独特的音轨标记**：国语音轨/粤语音轨/原生中字标记便于筛选
4. **标准命名规范**：英文标题命名严格，便于解析提取信息

### 作为源站的局限性
1. **普通用户只能发布原盘**：非Uploader无法发压制/WEB
2. **拒绝转发标记**：大量资源标记拒绝转发，转载需注意
3. **无PT-Gen/豆瓣集成**：资源描述中可能缺少结构化信息
4. **无标签系统**：无法通过标签筛选
5. **预告片仅支持优酷/土豆**：已过时的视频源

## 转发规则（关键）

### 全局禁转规则
> **所有 HDR、R2HD 后缀及其他官方资源（不论是否标注）均为禁转。**

这意味着：
1. **HDR 后缀资源**：所有 HDRoute 官方制作组（后缀为 HDR）的资源，无论是否标注拒绝转发，一律禁止转载
2. **R2HD 后缀资源**：所有 R2HD 制作组的资源同样禁止转载
3. **其他官方资源**：任何标注为官方的资源，不论是否勾选 `is_trans_denied`，均视为禁转
4. **is_trans_denied 标记**：作为额外保护层，部分资源会显式勾选拒绝转发

### 实际影响
- 从 HDRoute 采集数据时，必须检查制作组后缀是否为 HDR 或 R2HD
- 仅非官方/非 HDR/R2HD 后缀的资源才可能被转载
- 实际可转载资源范围非常有限

## 转载注意事项

作为仅源站（仅采集数据），从HDR采集数据时需注意：

1. **全局禁转检查**：所有 HDR/R2HD 后缀及官方资源均为禁转，不论是否标注
2. **拒绝转发标记**：`is_trans_denied=1` 的资源禁止转载（作为额外保护层）
3. **标题解析**：英文标题包含完整信息（片名/年份/载体/分辨率/编码/音轨/制作组），需解析提取
4. **中文标题处理**：HDR自动加【】，转载到其他站点时需去掉【】
5. **分类映射**：HDR的9分类需映射到目标站点分类
6. **媒介映射**：HDR的"Blu-ray原盘/Remux/HDTV Rip/BDRe/CD/其他/MP4"需映射到目标站点媒介
7. **分辨率映射**：注意ID不连续(4K2K=7)，且无SD档位
8. **音频字段**：HDR只有"最高音轨"（1个），目标站点可能需要所有音轨
9. **复选框属性**：国语/粤语/中字/DIY信息需转为目标站对应字段
10. **IMDb信息**：HDR存有得分和编号，可复用
11. **R2HD/HDR小组国语音轨**：规则禁止提取这些小组的国语音轨单独发布
