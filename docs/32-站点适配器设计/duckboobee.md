## 三、站点完整规则

> 来源：`rules.php`

### 3.1 总则

- 不要做管理员明文禁止的事情。
- 不允许发送垃圾信息。
- 一切作弊的账号会被封。
- 注册多个 March 账号的用户将被禁止。
- 不要把本站的种子文件上传到其他 Tracker。
- 第一次在论坛或服务器中的捣乱行为会受到警告，第二次永久封禁。
- 禁止涉及禁忌或敏感内容（如色情、敏感政治话题等）的资源。

### 3.2 账号保留规则

| 等级 | 规则 |
|------|------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上封存账号 | 不会被删除 |
| 封存账号用户 | 连续 **400 天**不登录 → 删除 |
| 未封存账号用户 | 连续 **150 天**不登录 → 删除 |
| 无流量账号（上传/下载都为 0） | 连续 **100 天**不登录 → 删除 |

### 3.3 上传总则

- 上传者必须对上传的文件拥有合法的传播权。
- 上传者必须保证上传速度与做种时间。**如果在其他人完成前撤种或做种时间不足 24 小时，或者故意低速上传**，上传者将会被警告甚至取消上传权限。
- **发布者将获得双倍的上传量。**

### 3.4 上传者资格

- **任何人都能发布资源**（user 等级即可）。
- 有些用户需要先在候选区提交候选。
- **游戏类资源**：只有上传员及以上等级，或管理组特别指定的用户才能自由上传。其他用户必须先在候选区提交候选。

### 3.5 允许的资源

- 高清视频（蓝光原碟/HD DVD 原碟/Remux/HDTV/Encode ≥720p、高清 DV）
- 标清视频（来源于高清媒介的标清编码 ≥480p、DVDR/DVDISO/DVDRip/CNDVDRip）
- 无损音轨（FLAC、Monkey's Audio 等）及 5.1 声道以上音轨（DTS、DTS-CD 镜像等）、评论音轨
- PC 游戏（必须为原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 3.6 禁止的资源

- 总体积 < 100MB（高清软件/文档、单曲专辑除外）
- 标清 upscale 或部分 upscale 视频
- 劣质标清：CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD
- RealVideo（RMVB/RM）、FLV
- 单独的样片（须与正片一起上传）
- 有损 MP3/WMA（未达 5.1 声道标准）
- 无正确 CUE 表单的多轨音频文件
- 硬盘版/高压版游戏、非官方游戏镜像、第三方 mod、小游戏合集、单独破解或补丁
- RAR 等压缩文件
- 重复（Dupe）资源
- **禁忌/敏感内容（含色情、敏感政治话题）**
- 损坏文件、垃圾文件（病毒、木马、网站链接、广告等）

### 3.7 Dupe 判定规则

**来源优先级**：Blu-ray/HD DVD > HDTV > DVD > TV

- 同视频高优先级版本使低优先级 dupe。
- 高清版本使标清版本 dupe。
- **动漫类特例**：HDTV 和 DVD 版本有相同优先级。
- 相同媒介、相同分辨率重编码：按发布组优先级判断，高优先级组版本使低/同级 dupe。
- **总会保留一个**当前最佳画质来源经重编码而成的 DVD5 大小（约 4.38 GB）版本。
- 基于无损截图对比，高质量版本使低质量版本 dupe。
- 不同区域、不同配音/字幕的 Blu-ray/HD DVD 原盘**不视为 dupe**。
- 每个无损音轨资源只保留一个版本，**分轨 FLAC 格式有最高优先级**。
- 旧版本断种 **45 天**以上或发布 **18 个月**以上 → 新版本不受 Dupe 约束。
- 新版本发布后，旧重复版本保留至断种。

### 3.8 资源打包规则

**允许打包**：
- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7 日内高清预告片
- 同一艺术家 MV（标清 MV 按 DVD 打包，不允许单曲单独发布；高清 MV 须分辨率相同）
- 同一艺术家音乐（5 张以上专辑方可打包；两年内专辑可单独发布；剔除站内已有或全部包括）
- 分卷发售动漫剧集/角色歌/广播剧
- 发布组打包发布的资源

**打包要求**：视频须来源媒介相同、分辨率相同、编码一致（预告片例外）；电影合集发布组也须统一。音频须编码格式一致。打包后视情况删除单独种子。

### 3.9 例外条款

- 允许 TV/DSR 体育标清视频
- 允许 <100MB 的高清软件和文档
- 允许 <100MB 的单曲专辑
- 允许 2.0 声道或以上的国语/粤语音轨
- 允许附带字幕、游戏破解/补丁、字体、包装扫描图（统一打包或统一不打包）
- 允许发布音轨时附带附赠 DVD 相关文件

### 3.10 种子信息填写规范

**标题命名**：
- 电影：`[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 电视剧：`[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 音轨：`[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]`
- 游戏：`[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]`

**外部信息**：电影和电视剧必须包含外部信息链接（如 IMDb 链接）。

**简介要求**：
- 电影/电视剧/动漫：必须含海报/横幅/封面；尽可能含截图、详细文件信息、演职员及剧情概要。
- 体育节目：不得在文字介绍/截图/文件名/文件大小/时长中泄露比赛结果。
- 音乐：必须含专辑封面和曲目列表。
- PC 游戏：必须含海报/封面；尽可能含截图。

**注意事项**：
- NFO 图写入 NFO 文件，不粘贴到简介。
- 管理员会编辑种子信息，请勿改变或去除管理员的修改。
- 种子信息不规范可能被删除。

## 四、促销规则

- 10% 概率 50% 下载、5% 免费、5% 2x 上传、3% 50%下载&2x 上传、1% 免费&2x 上传
- 体积 > 20GB 自动免费
- 蓝光/HD DVD 原盘自动免费
- 电视剧每季第一集免费
- 关注度高的种子由管理员设置促销
- 促销限时 7 天（2x 上传无时限）
- 发布 1 个月后自动永久 2x 上传
