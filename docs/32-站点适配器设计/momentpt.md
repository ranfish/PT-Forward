## 发布规则摘要

### 上传总则
1. 任何人都能发布资源
2. 部分用户需先在候选区提交候选
3. 做种时间不足 24 小时或故意低速上传将被警告甚至取消上传权限
4. 发布者获得双倍上传量

### 促销规则
- 随机促销：10% 概率 50%下载、5% 免费、5% 2x上传、3% 50%下载&2x上传、1% 免费&2x上传
- 大于 20GB 自动免费
- 发布 1 个月后自动永久 2x上传

### 盒子规则（2026.01.15 启用）
- 单种上传速度限制：500Mbps
- 72 小时内上传量上限：种子体积 × 3
- 需在控制面板手动登记盒子 IP
- VIP 及工作组以上等级不受限制

- **无 MediaInfo**

## 特殊说明

1. **纯图片站**：PT 圈罕见的图片专题站，禁止视频/音频/软件/游戏
2. **不适合做源站和发布站**：没有其它站点与瞬间类似，无法从其它 PT 站转发图片资源
3. **地区标签体系**：以国家/地区（CN/JP/FR/UK/US/KR/GER/OTHER）为标签核心
4. **无质量字段**：图片资源不涉及视频编码/音频编码/分辨率等传统质量维度
5. **双 mode 标签**：mode=4 有 20 个标签（含明星/樱花妹/可商用/OTHER），mode=5 有 15 个标签
6. **盒子规则完善**：500Mbps 限速 + 72h 分享倍率限制
