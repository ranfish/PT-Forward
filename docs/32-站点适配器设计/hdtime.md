## 二、转发规则与标题命名规范

> 来源：`forums.php?action=viewtopic&forumid=1&topicid=83`

### 2.1 转发规则

1. 个人会员未经许可**不得使用 HDTime/PadTime** 等标识或后缀发布作品
2. 转发资源**不得随意去掉或篡改**原创小组的标识、后缀（如 CHD/CMCT）
3. **严禁转发**其他站点的**独占、禁转、限转**（未解除限制前）资源
4. 禁止随意修改原始文件、文件名或自行增减内容后重新做种发布
5. 发布前先搜索站内是否有重复资源
6. 严格遵守命名及格式规范

### 2.2 标题命名规范

**视频/电影类**（含纪录片、动漫、综艺、体育、MV）：
- 主标题：`英文片名 版本信息 上映(发行)年代 发布说明 原始媒介 分辨率 音频信息 视频编码-制作人/发布组`
- 副标题：`中文片名(日韩文片名) [附加说明]`
- 例：`The Dark Knight 2008 PROPER 720p BluRay x264-SiNNERS` / `蝙蝠侠:黑暗骑士 [导演剪辑版] [国/英] [修正版]`

**剧集类**：
- 主标题：`英文剧集名 上映(发行)年代 季数集数 发布说明 分辨率 原始媒介 音频信息 视频编码-制作人/发布组`
- 副标题：`中文剧集名(日韩文剧集名) (季数 集数) [附加说明]`
- 例：`Prison Break S04E01 PROPER 720p HDTV x264-CTU` / `越狱 (第四季 第1集) [修正版]`

**音乐类**：
- 主标题：`英文艺术家名 - 英文专辑名 版本信息 发行年代 发布说明 音频编码-发布组`
- 副标题：`中文艺术家名 - 中文专辑名 版本信息 [附加说明]`
- 例：`Enya - And Winter Came Deluxe Edition 2008 APE` / `恩雅 - 冬季降临 [豪华收藏版]`

### 2.3 内容格式规范

- **视频电影类**：必须含海报/横幅/封面；尽可能含截图、详细文件信息（年代/国家/导演/主演/语言/字幕/格式/时长/分辨率等）
- **体育类**：不得在文字介绍/截图/文件名/文件大小/时长中泄露比赛结果
- **音乐类**：必须含专辑封面和曲目列表

## 三、站点完整规则

> 来源：`rules.php`（标准 NexusPHP 模板，与 LemonHD/三月高度一致）

### 3.1 总则

- 不要做管理员明文禁止的事情。
- 不允许发送垃圾信息。
- 一切作弊的账号会被封。
- 注册多个 HDTIME 账号将被禁止。
- 不要把本站种子文件上传到其他 Tracker。
- 第一次捣乱行为受警告，第二次永久封禁。

### 3.2 账号保留规则

| 等级 | 规则 |
|------|------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上封存 | 不删除 |
| 封存账号 | 连续 **400 天**不登录 → 删除 |
| 未封存账号 | 连续 **150 天**不登录 → 删除 |
| 无流量账号 | 连续 **100 天**不登录 → 删除 |

### 3.3 上传总则

- 上传者必须对上传的文件拥有合法的传播权。
- 做种时间不足 **24 小时**或故意低速上传 → 警告甚至取消上传权限。
- **发布者获得双倍上传量。**

### 3.4 上传者资格

- 任何人都能发布资源。
- 有些用户需先在候选区提交候选。
- **游戏类资源**：只有上传员及以上等级才能自由上传，其他用户须先候选。

### 3.5 允许的资源

- 高清视频（蓝光原碟/HD DVD 原碟/Remux/HDTV/Encode ≥720p、高清 DV）
- 标清视频（来源于高清媒介的标清编码 ≥480p、DVDR/DVDISO/DVDRip/CNDVDRip）
- 无损音轨（FLAC、Monkey's Audio 等）及 5.1 声道以上音轨
- PC 游戏（必须为原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 3.6 禁止的资源

- 总体积 < 100MB（高清软件/文档、单曲专辑除外）
- 标清 upscale 视频
- 劣质标清：CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD
- RealVideo（RMVB/RM）、FLV
- 单独的样片
- 有损 MP3/WMA（未达 5.1 声道标准）
- 无正确 CUE 表单的多轨音频文件
- 硬盘版/高压版游戏、非官方游戏镜像、第三方 mod
- RAR 等压缩文件
- 重复（Dupe）资源
- **禁忌/敏感内容（含色情、敏感政治话题）**
- 损坏文件、垃圾文件

### 3.7 Dupe 判定规则

**来源优先级**：Blu-ray/HD DVD > HDTV > DVD > TV

- 高优先级来源使低优先级 dupe；高清使标清 dupe。
- **动漫类特例**：HDTV 和 DVD 有相同优先级。
- 相同媒介/分辨率重编码按发布组优先级判断。
- **总会保留一个**DVD5 大小（约 4.38 GB）版本。
- 不同区域/不同配音/字幕的原盘**不视为 dupe**。
- 无损音轨只保留一个版本，**分轨 FLAC 优先级最高**。
- 旧版本断种 **45 天**或发布 **18 个月**以上 → 新版本不受 Dupe 约束。
- 新版本发布后，旧重复版本保留至断种。

### 3.8 资源打包规则

**允许打包**：电影合集、整季剧集/综艺/动漫、同专题纪录片、7 日内预告片、同艺术家 MV/音乐、分卷动漫/角色歌/广播剧、发布组打包。

**要求**：视频须来源媒介相同、分辨率相同、编码一致；电影合集发布组须统一。音频须编码格式一致。

### 3.9 促销规则

**随机促销**：

| 概率 | 促销类型 |
|------|----------|
| 10% | 50% 下载 |
| 5% | 免费 |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传 |

**自动促销**：体积 > 20GB 自动免费；蓝光/HD DVD 原盘自动免费；电视剧每季第一集免费。
**时限**：除"2x 上传"无时限外，其余限时 **7 天**。发布 1 个月后自动永久 2x 上传。

## 四、适配器设计注意事项

| 特点 | 说明 |
|------|------|
| mode=4 后缀 | 字段名带 `[4]` 后缀 |
| 无分辨率/来源/地区 | 分辨率通过标题传递，无下拉框 |
| 新一代编码 | 支持 AV1(15)、VVC/H.266(16)、AVS3(17)、Audio Vivid/AV3A(14) |
| Blu-Ray原盘分类 | 424 为独立分类而非质量选项 |
| x264-10bit | 独立编码选项(11)，需区分 |
| HEVC 值=12 | 非标准编号 |
| 四来源 PT-Gen | imdb/douban/bangumi/indienova |
| 认领系统 | 每用户最多 1000 种子 |
| Cloudflare 防护 | 需要 cf_clearance cookie |
| 候选制 | 导航栏有"候选"入口，游戏类需上传员等级 |
