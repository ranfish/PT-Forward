## 站点特色

- **原盘特效站**：以原盘资源为主，适合做源站下载资源
- **不适合做发布站**：用户等级为User时无直接发布权限，需走候选区
- **规则与学校站高度相似**：上传规则、Dupe规则、打包规则、促销规则几乎完全一致
- **与学校站的关键差异**：
  - **无 H&R**：学校站有完整 H&R 机制（10天/20h/≥10次ban），阳光站完全没有
  - **无认领系统**：学校站有认领规则（20人/100种/月300h），阳光站无
  - **无官组禁止选择**：学校站禁止非管理组选 BTSCHOOL/Zone/BtsHD/BtsTV/BtsPAD，阳光站无此限制说明
  - **有管理组退休待遇**：上传员→养老族/VIP，管理员→养老族/VIP（学校站无此规则）
  - **标题格式含中文前缀**：`[中文名] 英文名...`，学校站中文名在副标题
- **游戏类限制**：与学校站相同，仅上传员及以上可直接发布
- **促销自动**：随机促销 + >20GB自动Free + 蓝光原盘自动Free + 1个月后永久2X
- **发布者双倍上传量**

## 标题命名规范（来自rules.php）

### 电影类
- `[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 例：`蝙蝠侠:黑暗骑士 The Dark Knight 2008 PROPER 720p BluRay x264-SiNNERS`

### 剧集类
- `[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 例：`越狱 Prison Break S04E01 PROPER 720p HDTV x264-CTU`

### 音轨类
- `[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]`
- 例：`恩雅 - 冬季降临 Enya - And Winter Came 2008 FLAC`

### 游戏类
- `[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]`
- 例：`红色警戒3:起义时刻 Command And Conquer Red Alert 3 Uprising-RELOADED`

### 副标题规则
- 不要包含广告或求种/续种请求

### 外部信息
- 电影和电视剧**必须**包含外部信息链接（如IMDb链接）

### 简介要求
- **电影/电视剧/动漫**：必须包含海报/横幅/封面；尽可能包含截图、文件详情（格式/时长/编码/码率/分辨率/语言/字幕）、演职员及剧情概要
- **体育节目**：请勿在文字介绍或截图/文件名/文件大小/时长中泄漏比赛结果
- **音乐**：必须包含专辑封面和曲目列表
- **PC游戏**：必须包含海报/封面；尽可能包含截图
- NFO图请写入NFO文件，不要粘贴到简介里

## 发布规则

### 禁止内容
- 总体积小于100MB的资源
- 标清upscale视频
- CAM/TC/TS/SCR/DVDSCR/R5/HalfCD
- RMVB/RM/flv文件
- 有损MP3/有损WMA（未达5.1声道标准）
- RAR等压缩文件
- 硬盘版/高压版游戏
- 色情/敏感政治内容

### Dupe规则
- 来源媒介优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 动漫类HDTV与DVD同优先级（特例）
- 断种45日或发布18个月以上可重发
- 官方组发布后同版本视为Dupe

### 资源打包规则（试行）
- 与学校站完全一致
- 允许打包：电影合集/整季剧/专题纪录片/7日内预告片/同艺术家MV/同艺术家音乐(≥5张)/动漫分卷/发布组打包
- 打包约束：相同媒介/分辨率/编码，电影合集须发布组统一

### 账号保留
- Veteran User及以上：永久保留
- Elite User及以上封存后：不删除
- 封存账号：400天未登录删除
- 未封存账号：150天未登录删除
- 无流量用户：14天后封禁

## 管理组退休待遇

### 上传员 (Uploaders)
- **养老族 (Retiree)**：升职1年以上 + 上传≥200个种子资源（特殊情况由管理组投票）
- **VIP**：升职6个月以上 + 上传≥100个种子资源
- **其他**：成为 Extreme User（若条件满足 Extreme User 及以上则成为 Nexus Master）

### 管理员 (Moderators)
- **养老族 (Retiree)**：升职1年以上 + 参加至少2次站务组正式会议 + 参与规则/答疑修订
- **VIP**：若不满足养老族条件，可无条件成为 VIP

### 总管理员 (Administrators) 及以上
- 可直接成为养老族 (Retiree)

## 字幕区规则

- 允许格式：srt / ssa / ass / cue / zip / rar
- Vobsub格式（idx+sub）或其他格式须打包为 zip/rar
- 禁止上传 lrc 歌词等非字幕/cue 文件
- 不合格字幕（不匹配/不同步/打包错误/编码错误/重复等）直接删除，上传者扣100魔力
- 举报不合格字幕奖励50魔力

## 转载注意事项

1. **仅源站角色**：本站不适合做发布站，仅适合作为源站下载资源
2. **无发布权限**：User等级无直接发布权限，无法获取完整表单结构
3. **规则与学校站高度相似**：几乎完全相同的上传/Dupe/打包/促销规则，可能是同一套模板
4. **需升级后才能分析**：需达到发布权限等级后才能获取完整upload.php表单数据
5. **标题格式含中文前缀**：标题格式为`[中文名] 英文名...`，中文名在方括号中放在最前面
6. **与学校站的关键差异**：无 H&R、无认领系统、有管理组退休待遇、标题含中文前缀
