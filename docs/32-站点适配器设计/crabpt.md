## 发布规则

### 绝对禁止的资源

1. **他站禁止转载资源** — 举报奖励 2w 蟹币，违规者第一次警告，第二次 ban 号
2. **9KG 资源** — 美国 NC-17 / 香港 III 级 / 日本 R18 / 韩国 18 分级及露点资源
3. **质量较差的视频** — CAM/TC/TS/SCR/R5/HalfCD/MiniSD/MNHD/RMVB/RM/flv（电影类）
4. **重复资源** — 同一来源完全相同的资源，仅保留较早发布的
5. **国内正在上映的电影**
6. **其他站点规则中不允许的资源**

### 限制发布的资源

1. **BT 转载资源** — 仅允许种审组审核同意后发布
2. **已完结电视剧** — 必须以合集形式发布，限制单集

### 允许的资源

- 高清视频（720p+）：完整蓝光原盘/HD DVD/remux/HDTV/重编码
- 标清视频（仅限来源于高清媒介的重编码，至少 480p）
- 无损音轨（FLAC/APE 等）+ cue
- 5.1 声道及以上音轨
- PC 游戏原版光盘镜像
- 7 日内高清预告片
- 高清相关软件和文档

### 禁止的资源

- < 100MB（高清软件/单曲专辑例外）
- 标清 upscale
- RMVB/RM/flv
- 单独的样片
- < 5.1 声道的有损 MP3/WMA
- 无正确 cue 的多轨音频
- 硬盘版/高压版游戏
- RAR 压缩文件
- 损坏文件/垃圾文件/广告

### 重复（Dupe）判定规则

- 来源媒介优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 高清版本使标清版本成为 Dupe
- 动漫类 HDTV 和 DVD 同优先级（特例）
- 同媒介同分辨率的重编码按发布组优先级判定
- 保留一个 DVD5 大小的最佳画质重编码版本
- 不同配音/字幕的原盘不视为 Dupe
- 无损音轨原则上只保留一个版本，分轨 FLAC 最高优先级
- **断种 45 天**或**发布 18 个月**以上可发布新版本
- 新版本发布后旧版本保留至断种

### 资源打包规则（试行）

允许打包：
- 套装高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7 日内高清预告片
- 同一艺术家 MV（标清按 DVD 打包，高清同分辨率）
- 同一艺术家音乐（≥ 5 张专辑，两年内可单独发布）
- 分卷发售动漫/角色歌/广播剧
- 发布组打包

打包约束：相同媒介/分辨率/编码/发布组（预告片例外）

## 标题命名规范

### 基本原则

- 主标题不添加中文
- 无英文名使用拼音（Adult JAV 使用原文）
- 副标题不使用全角标点
- 禁止添加 MKV/MP4 等文件格式
- 禁止特殊符号 `!;.#$%:`
- DVD 注明 D5 或 D9
- 音频标记声道（2.0/5.1/7.1）
- 多音轨以高码率优先命名
- EAC3 = DDP，AC3 = DD
- 多音轨标注 2Audios/3Audios

### 电影类

```
英文名称 年份 来源 分辨率 视频编码 音频编码-小组名称
```

例：`Lisa Frankenstein 2024 BluRay 1080p x264 DTS-HD MA 5.1-MTeam`

### 电视剧类

```
英文名称 年份 季集 分辨率 来源 视频编码 音频编码-小组名称
```

例：`Best Choice Ever 2024 S01E11-E12 1080p WEB-DL x264 AAC 2.0-QHstudIo`

### 动画类（原盘）

```
英文名称 年份 分辨率 介质 视频编码(统一BluRay) 音频编码-小组名称
```

### 动画类（Encode）

```
英文名称 年份 分辨率 媒介(BDRip) 视频编码 音频编码-小组名称
```

### 动画类（WEB）

单集：`英文名称 年份 季集 分辨率 来源 视频编码 音频编码-小组名称`

合集：`英文名称 年份 季 分辨率 来源 视频编码 音频编码-小组名称`

### 音乐类

```
艺人名称 - 专辑名称 年份 音频编码 整轨/分轨-小组名称
```

> 体积 > 100G 的音乐种子需先联系管理组取得发布码。

### 游戏类

```
平台 类型 英文名-小组/出品公司
```
