## 三、发布规则要点

### 3.1 上传资格

- **任何人都能发布资源**（user 等级即可）。
- 游戏类资源需**上传员**及以上等级，或管理组特别指定，否则须先候选。

### 3.2 标题命名

- **电影**：`[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组`
- **电视剧**：`[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组`
- **音轨**：`[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组]`
- **游戏**：`[中文名] 名称 [年份] [版本] [发布说明][-发布组]`

### 3.3 允许的资源

- 高清视频（蓝光原碟/Remux/HDTV/Encode ≥720p）
- 标清视频（来源于高清媒介的标清编码 ≥480p、DVDR/DVDISO/DVDRip）
- 无损音轨（FLAC、APE 等）及 5.1 声道以上音轨
- PC 游戏（必须原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 3.4 禁止的资源

- 总体积 < 100MB（高清软件/文档、单曲专辑除外）
- 标清 upscale 视频
- 劣质标清：CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD
- RealVideo（RMVB/RM）、FLV
- 有损 MP3/WMA（未达 5.1 声道标准）
- 硬盘版/高压版游戏、非官方镜像、第三方 mod
- RAR 压缩文件
- 重复资源（dupe）
- 禁忌/敏感内容（含 9KG/色情/成人内容，**本软件禁止下载和发布**）
- 损坏文件、垃圾文件

### 3.5 Dupe 判定规则（按来源优先级）

**来源优先级**：Blu-ray/HD DVD > HDTV > DVD > TV

- 同媒介同分辨率重编码：按发布组优先级判断（参考 Scene & Internal 规范）
- 但**总会保留一个 DVD5 大小（4.38GB）的版本**
- 高优先级来源使低优先级 dupe
- 高清使标清 dupe（动漫类 HDTV/DVD 同级例外）
- 来自不同区域、不同配音/字幕的原盘**不视为 dupe**
- 无损音轨只保留一个版本（分轨 FLAC 优先级最高）
- 旧版本断种 45 日或发布 18 个月以上，新版本不受 dupe 约束

### 3.6 做种要求

- 做种时间不足 **24 小时**或故意低速上传会被警告甚至取消上传权限。
- 发布者获得**双倍上传量**。

---

## 四、促销规则

- 10% 概率 50% 下载、5% 免费、5% 2x 上传、3% 50%下载&2x 上传、1% 免费&2x 上传
- 体积 > 20GB 自动免费
- 蓝光/HD DVD 原盘自动免费
- 电视剧每季第一集免费
- 促销限时 7 天（2x 上传无时限）
- 发布 1 个月后自动永久 2x 上传

## 五、账号保留规则

| 等级 | 规则 |
|------|------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上 | 封存账号后不删除 |
| 封存账号 | 连续 400 天不登录删除 |
| 未封存账号 | 连续 150 天不登录删除 |
| 无流量账号 | 连续 100 天不登录删除 |

## 六、适配器设计注意事项

| 特点 | 说明 |
|------|------|
| mode=4 后缀 | 字段名带 `[4]` 后缀 |
| HEVC≠H.265 | 编码有两个独立选项（值2和值3），需注意区分 |
| 无音频编码 | 音频信息通过标题传递 |
| 音乐标签丰富 | 6 个音乐相关标签需正确映射 |
| 游戏类限制 | 非上传员需先走候选流程 |
| dupe 优先级 | 来源媒介优先级明确（Blu-ray > HDTV > DVD > TV） |
