## 一、发布页面表单字段分析

### 1.1 基础信息字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| `file` | file | ✓ | 种子文件 |
| `name` | text | - | 标题 |
| `small_descr` | text | - | 副标题 |
| `url` | text | - | IMDb 链接（`data-pt-gen="url"`） |
| `nfo` | file | - | NFO 文件 |
| `descr` | textarea | ✓ | 简介（BBCode） |
| `uplver` | checkbox | - | 匿名发布（value="yes"） |

### 1.2 PT-Gen 集成

支持多来源：IMDb + Douban + Bangumi + Indienova + PT-Gen（`data-pt-gen` 出现在 `url` 和 `pt_gen` 属性中）。

### 1.3 双区域发布（关键特色）

Farmm 有 **种子区（mode=4）** 和 **特别区（mode=5）** 两个发布区域，通过两个 `<select name="type">` 互斥切换（`onchange="disableother()"`）。

**种子区**（`id="browsecat"`, `data-mode='4'`）→ `relation="mode_4"`
**特别区**（`id="specialcat"`, `data-mode='5'`）→ `relation="mode_5"`

两个区域有 **完全不同的分类体系和质量字段值**。

### 1.4 缺失字段

- **`source_sel[5]`**：特别区无来源字段
- 特别区部分字段值含多个"待定"占位符（codec_sel[5] 值 20-24）

## 四、关键适配器设计要点

### 4.1 双区域互斥发布

种子区和特别区通过两个 `<select name="type">` 互斥切换（`disableother()`）。适配器需根据内容类型选择正确的区域。

### 4.2 source_sel = 地区

`source_sel[4]` 在 Farmm 表示地区（内地/港台/日韩/欧美/印度/泰国），非来源媒介。特别区无此字段。

### 4.3 processing_sel 语义按区域切换

| 区域 | processing_sel 含义 | 值 |
|------|---------------------|-----|
| 种子区（mode=4） | **教育年级**（一年级到大学） | 34-46 |
| 特别区（mode=5） | **MPAA/TV 内容分级** | 18-24 |

这是 **同名字段在不同区域完全不同含义** 的典型案例。

### 4.4 编码细分

区分 H.265/HEVC vs X265 vs HEVC、H.264/AVC vs X264，适配器需根据源站编码名精确匹配。

### 4.5 Cloudflare 防护

站点使用 Cloudflare（cookie 含 `cf_clearance`），需参考 `docs/31-模块设计决策记录.md §29` 的 TLS 指纹绕过方案。

### 4.6 儿童教育特色

特别区专门面向儿童内容，分类、内容分级（MPAA/TV）都是围绕此主题设计。适配器转发儿童内容时应优先使用特别区。

### 4.7 站点仍在建设中

特别区多个"待定"占位（编码 5 个、分类 7 个），说明站点功能仍在完善中。适配器需定期更新字段映射。

## 六、站点规则（rules.php）

> 以下内容来源于 2026-04-23 采集的 rules.php 页面（直连 200，无需 Cookie 认证）。

### 6.1 账号保留规则

| 条件 | 期限 |
|------|------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上封存账号 | 不删除 |
| 封存账号连续不登录 | **400 天**删除 |
| 未封存账号连续不登录 | **150 天**删除 |
| 无流量用户连续不登录 / 注册满 | **60 天**封禁 |
| 被封禁账号连续不登录 | 180 天物理删除 |

### 6.2 下载规则 — 种子促销

#### 6.2.1 随机促销（种子上传后自动随机设定）

| 概率 | 促销类型 |
|------|----------|
| 10% | 50%下载 |
| 5% | 免费 |
| 5% | 2x上传 |
| 3% | 50%下载 & 2x上传 |
| 1% | 免费 & 2x上传 |

#### 6.2.2 自动促销

| 条件 | 促销类型 |
|------|----------|
| 文件总体积 > **100GB** | 免费 |
| Blu-ray Disk / HD DVD 原盘 | 免费 |
| 电视剧等每季第一集 | 免费 |
| 关注度高的种子 | 管理员定夺 |

#### 6.2.3 促销时限

- 除 **2x上传** 外，其余类型限时 **7 天**（自种子发布时起）
- **2x上传** 无时限
- 所有种子发布 **1 个月**后自动永久成为 **2x上传**

### 6.3 上传规则

#### 6.3.1 上传总则

- 发布者获得 **双倍上传量**（非 5 倍，与 LemonHD 不同）
- 必须保证上传速度与做种时间（不足 24 小时将警告甚至取消上传权限）

#### 6.3.2 上传者资格

- 任何人都能发布资源
- 游戏类资源需 **上传员** 及以上等级或管理组特别指定

#### 6.3.3 允许的资源

- 高清视频（完整媒介/Remux/HDTV/重编码≥720p/DV）
- 标清视频（来源于高清媒介的重编码≥480p / DVDR / DVDRip）
- 无损音轨（FLAC、Monkey's Audio 等）及 5.1 声道以上音轨
- PC 游戏（原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

#### 6.3.4 不允许的资源

- 总体积 < **100MB** 的资源
- 标清 upscale 视频
- CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD 等低质量
- RealVideo（RMVB/RM）、flv 文件
- 未达 5.1 声道的有损音频（MP3/WMA）
- 无正确 cue 的多轨音频
- 硬盘版/高压版游戏、非官方镜像、第三方 mod
- RAR 等压缩文件
- 重复（dupe）资源
- 色情/敏感内容
- 损坏文件/垃圾文件

### 6.4 Dupe 判定规则

与 LemonHD 完全一致（标准 NexusPHP dupe 模板）：

- 来源优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 高清版本使标清版本 dupe
- 动漫类 HDTV 和 DVD 同优先级（特例）
- 同媒介同分辨率重编码：按发布组优先级判定
- 保留一个 DVD5 大小的重编码版本
- 不同区域/不同配音/字幕的原盘不算 dupe
- 无损音轨只保留一个版本（分轨 FLAC 优先级最高）
- 新版本修复旧版本错误/来源更好 → 允许发布
- 旧版本断种 **45 天**以上或发布 **18 个月**以上 → 新版本不受 dupe 约束

### 6.5 资源打包规则

## 七、全局转载策略影响分析（§30.5）

### 7.1 禁止国内影视资源

Farmm **禁止国内影视资源（含港台）**，仅教育区不受此限。但以下国内资源**不在此限**：
- MV、综艺、纪录、体育、动漫
- 电影合集（按合集形式）
- 电视剧合集（按合集形式）

**转发影响**：源站为国内影视内容的种子，**不能转发到 Farmm**（除非属于上述例外类别或教育区分类）。

### 7.2 禁转标签

Farmm 有 **禁转** 标签（值=1）。带有禁转标签的种子不应被转发出去，同样，源站带禁转/独占/谢绝转载标签的种子也不应转发到 Farmm。

### 7.3 制作组映射

仅 FFWEB + Other。绝大多数源站制作组在 Farmm 都映射为 Other。

### 7.4 编码细分

Farmm 区分 H.265/HEVC/X265/HEVC 和 H.264/AVC/X264，适配器需根据标题中的编码标识精确匹配：
- 标题含 `x265` → X265（值=3）
- 标题含 `x264` → X264（值=4）
- 标题含 `HEVC` 但无 `x265` → H.265/HEVC（值=1）或 HEVC（值=5）
- 标题含 `AVC` 但无 `x264` → H.264/AVC（值=2）
