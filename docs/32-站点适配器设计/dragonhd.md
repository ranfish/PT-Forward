## 一、总则

### 1.1 账号保留规则

1. Veteran User 及以上等级永远保留
2. Elite User 及以上等级封存账号后不会被删除
3. 封存账号连续 **400 天**不登录 → 删除
4. 未封存账号连续 **150 天**不登录 → 删除
5. 零流量用户连续 **100 天**不登录 → 删除

### 1.2 通用规则

- 天大地大，站长最大
- 一切作弊账号被封
- 禁止注册多个账号
- 禁止将本站种子上传到其他 Tracker
- 第一次捣乱行为警告，第二次永久封禁

## 二、促销规则

### 2.1 随机促销（种子上传后系统自动随机设定）

| 概率 | 促销类型 |
|------|---------|
| 10% | 50% 下载 |
| 5% | 免费 |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传 |

### 2.2 自动免费

- 文件总体积 **> 20GB** → 自动免费
- **Blu-ray Disk / HD DVD 原盘** → 自动免费
- 电视剧等 **每季第一集** → 自动免费
- 关注度高的种子 → 管理员手动促销

### 2.3 促销时限

- 除"2x 上传"外，其余类型促销限时 **7 天**
- "2x 上传"无时限
- 所有种子发布 **1 个月后** → 自动永久成为 **2x 上传**
- 不定期全站免费

## 三、上传规则

### 3.1 上传总则

- 上传者必须对上传的文件拥有合法传播权
- 做种时间不足 **24 小时**或故意低速上传 → 警告甚至取消上传权限
- 发布者获得 **双倍上传量**
- 违规但有价值的资源 → 告知管理组，可能破例

### 3.2 上传者资格

- 任何人都能发布资源
- 部分用户需先在候选区提交候选
- **游戏类资源**：仅上传员及以上等级或管理组指定用户可自由上传

### 3.3 允许的资源

- 高清视频：蓝光原碟/HD DVD/remux/HDTV/重编码（≥720p）/高清 DV
- 标清视频：高清媒介的标清重编码（≥480p）/DVDR/DVDISO/DVDRip/CNDVDRip
- 无损音轨（FLAC/Monkey's Audio 等）+ cue 表单
- 5.1 声道及以上音轨（DTS/DTS-CD 镜像等）+ 评论音轨
- PC 游戏（必须原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 3.4 不允许的资源

- 总体积 < **100MB**（例外：高清软件文档、单曲专辑）
- 标清 upscale 视频
- 低质量标清：CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD
- RealVideo/RMVB/RM/FLV
- 单独样片（须与正片一起上传）
- < 5.1 声道有损音频（MP3/WMA 等）
- 无正确 cue 的多轨音频
- 硬盘版/高压版游戏、非官方镜像、第三方 mod
- RAR 压缩文件
- Dupe 资源
- 禁忌/敏感内容
- 损坏文件/垃圾文件

## 四、Dupe 判定规则

### 4.1 来源媒介优先级

**Blu-ray / HD DVD > HDTV > DVD > TV**

- 高优先级版本使低优先级版本被判定为 Dupe
- 高清版本使标清版本被判定为 Dupe
- **动漫特例**：HDTV 和 DVD 版本优先级相同

### 4.2 同媒介同分辨率重编码

- 按发布组优先级判定（参考 "Scene & Internal, from Group to Quality-Degree"）
- **DVD5 保留**：总会保留一个当前最佳画质来源经重编码而成的 DVD5 大小（≈4.38GB）的版本
- 基于无损截图对比，高质量版本使低质量版本被视为 Dupe

### 4.3 豁免条款

- 不同区域/不同配音/字幕的 Blu-ray/HD DVD 原盘 → 不算 Dupe
- 每个无损音轨原则上只保留一个版本（分轨 FLAC 优先级最高）
- 新版本修复旧版本错误或有更高质量来源 → 允许发布，旧版本被视为 Dupe
- 旧版本连续断种 **45 日以上**或发布 **18 个月以上** → 不受 Dupe 约束
- 新版本发布后，旧版本保留至断种

## 五、资源打包规则（试行）

### 5.1 允许打包

- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7 日内高清预告片
- 同一艺术家 MV（标清按 DVD 打包，禁止单曲单独发布；高清分辨率须一致）
- 同一艺术家音乐（≥5 张专辑方可打包；两年内专辑可单独发布；剔除站内已有资源或全部包含）
- 分卷发售动漫剧集/角色歌/广播剧
- 发布组打包发布的资源

### 5.2 打包一致性要求

- 视频：相同媒介类型/相同分辨率/相同编码格式（预告片例外，电影合集须发布组统一）
- 音频：编码格式一致（如全为分轨 FLAC）
- 打包发布后视情况删除相应单独种子

## 六、例外规则

- 允许 TV/DSR 来源体育类标清视频
- 允许 < 100MB 高清相关软件和文档
- 允许 < 100MB 单曲专辑
- 允许 2.0 声道及以上国语/粤语音轨
- 允许附带字幕/游戏破解与补丁/字体/包装扫描图（须统一打包或统一不打包）
- 允许发布音轨时附带附赠 DVD 相关文件

---

## 七、字幕区规则

### 7.1 基本规则

- 允许格式：srt / ssa / ass / cue / zip / rar
- Vobsub（idx+sub）或合集须打包为 zip/rar
- 禁止 lrc 歌词或非字幕/cue 文件

### 7.2 不合格判定

与种子不匹配/不同步/打包错误/含无关文件/编码错误/cue 错误/语种标识错误/标题命名不明确/重复/被举报证实

### 7.3 奖惩

- 举报不合格字幕确认后 → 奖励举报者 **50 魔力值**
- 不合格字幕上传者 → 扣除 **100 魔力值**
- 恶意上传或恶意举报 → 额外扣魔力值甚至警告

## 九、关键适配器设计要点

### 9.1 无标签系统

DragonHD 完全没有标签（tags）功能，是目前分析过唯一没有标签的站点。

### 9.2 极简字段

仅有 5 个质量字段（medium/codec/audiocodec/standard/team），无 source_sel、processing_sel。字段名不带 data-mode 后缀。

### 9.3 2K/1440p 分辨率

分辨率先项含 2K/1440p，适配器需支持此分辨率的映射。

### 9.4 繁体中文界面

站点界面使用繁体中文，但字段名仍为英文（NexusPHP 标准）。

### 9.5 无 PT-Gen/豆瓣

没有 PT-Gen 和豆瓣独立字段，影视信息只能通过 IMDb 链接或简介手动填写。
