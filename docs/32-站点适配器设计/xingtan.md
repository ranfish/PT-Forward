# 杏坛 站点适配器设计

## 站点信息

| 项目 | 内容 |
|------|------|
| 站点名称 | 杏坛 |
| 站点地址 | https://xingtan.one |
| 站点框架 | NexusPHP（深度定制，学术资源专题） |
| 特殊功能 | **综合专业学术资源交流平台**，只分享专业和学术资源，非影视/音乐站 |
| 发布页面 | upload.php |

**站点角色**: 目标站（发布站）。**仅支持学术/专业资源发布**，不接受影视/音乐/游戏资源。

### 1.5 缺失的质量字段

杏坛**完全没有**以下标准影视质量字段：

| 字段 | 状态 |
|------|------|
| `medium_sel` | ❌ 无 |
| `codec_sel` | ❌ 无 |
| `standard_sel` | ❌ 无 |
| `audiocodec_sel` | ❌ 无 |
| `team_sel` | ❌ 无 |

**原因**: 杏坛为学术资源平台，不涉及影视质量属性（无分辨率/编码/音频编码/制作组概念）。

## 二、杏坛特殊注意事项

### 2.1 学术资源专题站

杏坛是**综合专业学术资源交流平台**，只分享专业和学术资源。字段含义被重新定义：

| 标准 NexusPHP 字段名 | 杏坛实际含义 |
|----------------------|-------------|
| `source_sel` | 资源载体（视频/音频/书籍/文档/课件...） |
| `processing_sel` | 文件格式（MP4/PDF/EPUB/CAJ/UVZ...） |
| `type` (mode) | 学科分类（哲学/经济学/法学/医学...90+个） |
| `author` | 作者（新增字段） |
| `publisher` | 出版社（新增字段） |
| `year` | 年份（新增字段） |
| `ftype` | 文件类型（text输入，新增字段） |
| `isbn` | ISBN（新增字段） |
| `medium_sel` | ❌ 不存在 |
| `codec_sel` | ❌ 不存在 |
| `standard_sel` | ❌ 不存在 |
| `audiocodec_sel` | ❌ 不存在 |
| `team_sel` | ❌ 不存在 |

### 2.2 适配器无法通用

由于完全没有影视质量字段，杏坛**不能使用标准的字段映射逻辑**。适配器需要：
- 完全独立的映射表
- 将资源的学科领域映射到 mode 值
- 根据文件后缀自动匹配 processing_sel（文件格式）
- 根据资源类型匹配 source_sel（载体类型）
- 无需处理编码/分辨率/音频编码/制作组

### 2.3 与好学站 (HXPT) 的对比

| 特征 | 杏坛 | 好学 (HXPT) |
|------|------|------------|
| 定位 | 综合学术资源 | 纯教育（K12+高等教育） |
| 分类 | **90+个学科分类** | 7个年级段 |
| 媒介字段 | `source_sel`（10个载体） | `medium_sel`（10个载体） |
| 文件格式 | `processing_sel`（27个格式） | `codec_sel`（26个格式） |
| 年级 | ❌ 无 | `audiocodec_sel`（14个年级） |
| 教材版本 | ❌ 无 | `processing_sel`（27个版本） |
| 分辨率 | ❌ 无 | `standard_sel`（6个） |
| 制作组 | ❌ 无 | 4个 |
| 年份 | `year`（text输入） | `source_sel`（18个年份选项） |
| 标签 | 11个（通用标签） | 24个（含13个学科） |
| 特有字段 | author/publisher/ISBN | price/uplver |
| 匿名发布 | ❌ 无 | ✅ 有 |
| 学术专用格式 | CAJ/UVZ/DJVU | CAJ/UVZ/DJVU |

### 2.4 特殊文件格式

以下文件格式为学术资源特有：
- **CAJ** (18) — 中国知网专用格式
- **UVZ** (25) — 超星电子书专用格式
- **DJVU** (17) — 学术文献扫描常用格式

### 2.5 无匿名发布

杏坛 upload.php 中**没有 `uplver` 字段**，不支持匿名发布。

## 三、与其他站点对比

| 特征 | 杏坛 | 常规 NexusPHP PT 站 |
|------|------|---------------------|
| 定位 | **学术/专业资源** | 综合影视/音乐 |
| 分类 | **90+个学科** | 通常 5-15 个影视分类 |
| 来源字段 | **载体类型（10个）** | 通常 5-8 个视频来源 |
| 格式字段 | **文件格式（27个）** | 通常 5-10 个视频编码 |
| 分辨率 | ❌ 无 | 通常 5-7 个 |
| 音频编码 | ❌ 无 | 通常 6-22 个 |
| 制作组 | ❌ 无 | 通常 3-37 个 |
| 标签 | 11个（通用） | 通常 3-37 个 |
| 特有字段 | author/publisher/ISBN/year | IMDb/MediaInfo |
| 字段含义 | **部分重定义** | 标准 NexusPHP |
