## 一、总则与账号规则

### 1.1 账号保留规则

| 条件 | 保留规则 |
|------|----------|
| Veteran User 及以上等级 | 永远保留 |
| Elite User 及以上等级 | 封存账号后不会被删除 |
| 封存账号的用户 | 连续 **120天** 不登录将被删除 |
| 未封存账号的用户 | 连续 **60天** 不登录将被删除 |
| 无流量用户 | 连续 **365天** 不登录将被删除 |
| 新注册用户 | **30天** 无流量将被封禁 |

### 1.2 封禁规则

- 注册多个昆仑账号将被禁止
- 第一次捣乱行为警告，第二次永久无缘昆仑
- Peasant 如果在 **7天内** 仍未改善分享率，将被禁用

### 1.3 Peasant 降级条件

| 条件 | 降级为 Peasant |
|------|----------------|
| 下载量 >50GB 且分享率 <0.4 | ✓ |
| 下载量 >100GB 且分享率 <0.5 | ✓ |
| 下载量 >200GB 且分享率 <0.6 | ✓ |
| 下载量 >400GB 且分享率 <0.7 | ✓ |
| 下载量 >800GB 且分享率 <0.8 | ✓ |

## 二、促销规则

### 2.1 随机促销

| 概率 | 促销类型 |
|------|----------|
| 0% | 50% 下载 |
| **90%** | 免费(Free) |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 2% | 免费 & 2x 上传(2XFREE) |

**极高免费率**：90% 的种子上传后自动免费，是目前所有分析站点中最高的。

### 2.2 自动促销

| 条件 | 促销类型 |
|------|----------|
| 文件总体积 **>1GB** | 自动免费 |
| Blu-ray Disk、HD DVD 原盘 | 免费 |
| 电视剧等每季的**第一集** | 免费 |
| 关注度高 | 管理员设置促销 |

**注意**：>1GB 即自动免费，门槛极低（其他站通常为 >20GB 或 >30GB）。

### 2.3 促销时限

- 除了"2x 上传"以外，其余类型的促销限时 **5天**（自种子发布时起）
- "2x 上传"**无时限**

## 三、上传规则

### 3.1 上传总则

- 上传者必须对上传的文件拥有合法的传播权
- 做种时间不足 **24小时** 或故意低速上传，将被警告甚至取消上传权限
- 发布者将获得 **双倍上传量**

### 3.2 上传者资格

- 任何人都能发布资源
- 有些用户需要先在候选区提交候选
- **游戏类资源**：只有上传员及以上等级，或管理组特别指定用户，可自由上传；其他用户须候选

### 3.3 允许的资源

- 高清视频（蓝光原碟/remux/HDTV/重编码≥720p/高清DV）
- 标清视频（来源于高清媒介的标清重编码≥480p、DVDR/DVDISO、DVDRip、CNDVDRip）
- 无损音轨（FLAC、Monkey's Audio 等，含 cue 表单）
- 5.1声道或以上音轨（DTS、DTSCD 镜像等）、评论音轨
- PC 游戏（必须为**原版光盘镜像**）
- **7日内**发布的高清预告片
- 高清相关软件和文档

### 3.4 不允许的资源

- 总体积小于 **100MB**（高清软件/文档、单曲专辑除外）
- 标清 upscale 视频
- 劣质视频（CAM、TC、TS、SCR、DVDSCR、R5、R5.Line、HalfCD 等）
- RealVideo 编码（RMVB/RM）、flv 文件
- 单独的样片
- 未达到 5.1 声道的有损音频（MP3、WMA 等）
- 无正确 cue 表单的多轨音频
- 硬盘版/高压版游戏、非官方游戏镜像、第三方 mod、小游戏合集、单独破解或补丁
- RAR 等压缩文件
- 重复(dupe)资源、禁忌敏感内容、损坏文件、垃圾文件

### 3.5 Dupe 判定规则

**来源媒介优先级**：
```
Blu-ray/HD DVD > HDTV > DVD > TV
```

**详细规则**：
- 高清版本使标清版本判定为重复
- **动漫类**：HDTV 和 DVD 有**相同优先级**（特例）
- 同媒介同分辨率重编码：按发布组确定优先级
- 总会保留一个当前最佳画质来源经重编码而成的 **DVD5 大小**（4.38GB 左右）的版本
- 基于无损截图对比，高质量版本使低质量版本视为重复
- 跨区域不同配音/字幕的 Blu-ray/HD DVD 原盘不视为重复
- 无损音轨：每个资源只保留一个版本，**分轨 FLAC** 优先级最高

**Dupe 豁免**：
- 新版本没有旧版本已确认错误/画质问题，或新版本来源质量更好 → 允许发布
- 旧版本连续断种 **45日以上** 或已发布 **18个月以上** → 不受 Dupe 约束
- 新版本发布后，旧的重复版本保留至断种

### 3.6 资源打包规则（试行）

允许打包：
- 按套装售卖的高清电影合集
- 整季电视剧/综艺节目/动漫
- 同一专题纪录片
- **7日内**高清预告片
- 同一艺术家 MV（标清按 DVD 打包、高清需分辨率相同）
- 同一艺术家音乐（**5张以上**专辑方可打包；**两年内**专辑可单独发布）
- 分卷动漫剧集、角色歌、广播剧
- 发布组打包资源

打包要求：相同媒介、相同分辨率、编码一致、发布组统一

### 3.7 标题命名规则

- 电影：`[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 电视剧：`[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 音轨：`[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]`
- 游戏：`[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]`

### 3.8 例外规则

- 允许发布 TV/DSR 体育类标清视频
- 允许发布 <100MB 高清相关软件和文档
- 允许发布 <100MB 单曲专辑
- 允许发布 **2.0声道及以上**国语/粤语音轨
- 允许附带字幕、游戏破解补丁、字体、包装扫描图（统一打包或不打包）
- 允许发布音轨时附带附赠 DVD 文件

## 四、字幕区规则

- 允许格式：srt/ssa/ass/cue/zip/rar
- Vobsub(idx+sub)或其他格式/合集需打包上传
- 不允许 lrc 歌词或非字幕/cue 文件
- 不合格判定：不匹配/不同步/打包错误/编码错误/语种标识错误/命名不明确/重复
- 举报奖励 **50魔力**，上传者扣除 **100魔力**

## 六、关键适配器设计要点

### 6.1 processing_sel = 地区

`processing_sel` 在通常 NexusPHP 站中表示处理方式（Raw/Encode），但在 YHPP 中表示 **地区**（CN/HK/TW/US/JP 等）。适配器需按字段实际含义映射，不能按字段名假设。

### 6.2 媒介最全

19 个媒介是目前所有分析站点中最多的，UHD/BD 各四级 + 音乐媒介（黑胶/CD+VCD/CD+DVD/SACD）。

### 6.3 音频编码最全

23 个音频编码是目前最多的，含 ALAC/m4a/DSD/TTA 等少见格式。

### 6.4 标签丰富

19 个标签含 HDR 三级细分（HDR/DV/Atmos）和分辨率标签（4K/8K），适配器可精细标注。
