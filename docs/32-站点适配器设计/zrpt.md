## 站点特色

- **纪录片特色站**：分类以纪录片题材为主（军事纪实/生物自然/餐饮美食/人文记录/宇宙星球/音乐视频/体育运动/其它）
- **新站**：建站于2025年
- **全站Free生效中**：2025-09-04~2026-06-04（限速生效期间）
- **发布者双倍上传**：发布者获得双倍上传量
- **随机促销**：10%概率50%下载，5%概率免费，5%概率2x上传，3%概率50%下载&2x上传，1%概率免费&2x上传
- **大体积自动免费**：>20GB自动免费，Blu-ray/HD DVD原盘免费，电视剧每季第一集免费
- **促销时限**：除2x上传无时限外，其余促销限时7天；发布1个月后自动永久2x上传
- **9KG/成人内容禁止**：本软件禁止下载和发布此类内容
- **规则与织梦高度同模板**：上传规则/Dupe/打包规则与织梦站几乎完全一致

## 标题命名规范

> 从规则页面提取。

### 电影
```
[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称
```
示例：`蝙蝠侠:黑暗骑士 The Dark Knight 2008 PROPER 720p BluRay x264-SiNNERS`

### 电视剧
```
[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称
```
示例：`越狱 Prison Break S04E01 PROPER 720p HDTV x264-CTU`

### 音轨
```
[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]
```
示例：`恩雅 - 冬季降临 Enya - And Winter Came 2008 FLAC`

### 游戏
```
[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]
```
示例：`红色警戒3:起义时刻 Command And Conquer Red Alert 3 Uprising-RELOADED`

## 简介规范

### 必须包含
1. **海报/封面**（电影、电视剧、动漫、游戏必须）
2. **演职员名单 + 剧情概要**（电影、电视剧、动漫尽量包含）
3. **文件的详细情况**：格式、时长、编码、码率、分辨率、语言、字幕
4. **画面截图**（尽量包含）

### 音乐类
- 必须包含专辑封面和曲目列表

## Dupe判定规则

与织梦站规则高度一致：

| 规则 | 说明 |
|------|------|
| 媒介优先级 | Blu-ray/HD DVD > HDTV > DVD > TV |
| 高清优于标清 | 高清版本使标清版本被判定为重复 |
| 动漫特例 | HDTV和DVD版本有相同优先级 |
| 同媒介同分辨率 | 按发布组优先级确定 |
| DVD5保留 | 总会保留一个DVD5大小（4.38GB）的版本 |
| 截图对比 | 基于无损截图对比，高质量版本使低质量版本被视重复 |
| 不同区域原盘 | 不同配音/字幕的blu-ray/HD DVD不算重复 |
| 无损音轨 | 原则上只保留一个版本，分轨FLAC优先级最高 |
| 断种替换 | 旧版本连续断种45日或发布18个月以上，新版本不受约束 |

## 打包规则

与织梦站规则高度一致：

- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7日内高清预告片
- 同一艺术家MV（标清按DVD打包，单曲不单独发布；分辨率相同的HD MV）
- 同一艺术家音乐（5张以上方可打包，两年内可单独发布）
- 分卷动漫剧集/角色歌/广播剧
- 发布组打包
