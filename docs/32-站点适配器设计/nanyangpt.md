## 一、发布页面表单字段分析

### 1.1 基础信息字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| `cite_site` | text | - | **引用**：输入站内资源链接，一键引用（`id="cite_site"`） |
| `type` | select | ✓ | 类型/分类 |
| `file` | file | ✓ | 种子文件（id="torrent"） |
| `name` | text | - | 标题（规范填写，高级用户可见） |
| `small_descr` | text | - | 副标题 |
| `url` | text | - | IMDb 链接 |
| `dburl` | text | 建议 | **豆瓣链接**（字段名 `dburl`） |
| `nfo` | file | - | NFO 文件 |
| `descr` | textarea | ✓ | 简介（BBCode） |

### 1.2 缺失字段

以下常见字段在南洋PT **全部不存在**：

- **`medium_sel`**：无媒介下拉框
- **`codec_sel`**：无编码下拉框
- **`standard_sel`**：无分辨率下拉框
- **`audiocodec_sel`**：无音频编码下拉框
- **`team_sel`**：无制作组下拉框
- **`source_sel`**：无来源字段
- **`processing_sel`**：无处理方式字段
- **`uplver`**：无匿名发布选项
- **`pt_gen`**：无 PT-Gen 字段
- **标签系统**：无标签（tags）功能

**南洋PT是至今分析的所有站点中，发布页面质量字段最极简的站点之一**（与 Hamster/HUDBT 类似，无质量下拉框）。所有质量信息仅通过分类和标题传达。

### 1.4 引用功能（cite_site）

南洋PT有独特的 **站内引用** 功能：
- 字段 `cite_site`：输入待引用的资源链接
- 按钮 `cite_button`："一键引用"
- 目前仅支持站内引用

此功能允许从已有种子直接复制信息，简化发布流程。

## 二、关键适配器设计要点

### 2.1 无质量字段

南洋PT发布页面 **没有任何质量下拉框**（媒介/编码/分辨率/音频/制作组/标签全部缺失）。适配器只需填写：
- `type`（分类）
- `file`（种子文件）
- `name`（标题——需规范填写）
- `url`（IMDb 链接）
- `dburl`（豆瓣链接）
- `descr`（简介）

这使得南洋PT成为 **最容易适配的目标站** 之一。

### 2.2 禁止搬运蓝光原盘/REMUX

规则明确：**未经允许，禁止搬运发布蓝光原盘（包括DIY）及REMUX类资源**。需在论坛申请报备。适配器在转发此类资源时应跳过南洋PT，或标记为需人工审核。

### 2.3 Dupe 规则（关键）

- **电影类、纪录片类**：**不执行 dupe**——支持多家重编码
- **剧集类**：**执行 dupe**——同规格同来源剧集包只保留一版
- 剧集 dupe 标准：1080p H264/265、2160p H265 等规格只留一版
- 优先级：高码 > 低码、软字幕 > 硬字幕、含国配优先

### 2.4 WEB-DL 限制

- 仅接受国外流媒体源（Amazon、Netflix、iTunes 等）
- 不接受 WEB-DL 源的 DIY
- 原则上不接受 WEB-DL 源的压制作品
- 音轨封装仅允许特定组合（原始/原始+英配/原始+国配等）
- 当 1080p 存在时，不接受 720p

### 2.5 码率最低要求

- 720p：不低于 4000kbps
- 1080p：不低于 8000kbps
- 720p 以下原则上不允许发布（DVD 除外）

### 2.6 豆瓣链接

南洋PT有独立的豆瓣链接字段（`dburl`），标记为"建议"填写。适配器可利用此字段。

### 2.7 候选制

有候选系统（offers.php），部分用户需先提交候选。

### 2.8 标题规范

标题需规范填写（链接到论坛 topicid=20 的规范帖），如：
```
银翼杀手 Blade Runner 1982 Final Cut 720p HDDVD DTS x264-ESiR
```

## 四、适配器转发策略建议

### 4.1 转发前过滤

适配器在向南洋PT转发前应执行以下检查：

1. **蓝光原盘/REMUX**：跳过或标记需人工审核
2. **WEB-DL DIY**：跳过
3. **WEB-DL 压制**：原则上跳过
4. **720p 以下**：跳过（DVD 除外）
5. **码率过低**：720p < 4000kbps 或 1080p < 8000kbps → 跳过

### 4.2 转发时填写

仅需填写 `type`、`file`、`name`、`url`、`dburl`、`descr`，无需处理任何质量映射。

### 4.3 剧集 Dupe 检查

转发剧集类资源前，应检索站内是否已有同规格同来源的版本。
