## 一、发种规范

### 1.1 标题规则（影视类）

**电影**：`英文名 年份 分辨率 介质 视频编码 音轨编码-制作名称`
- 例：`Avengers Infinity War 2008 1080p BluRay x264 DTS-WiKi`

**电视剧**：`电视台名(可选) 英文名 年代(可选) 季数/集数 分辨率 介质 视频编码 音轨编码-制作组名称`

**动漫组发布**：`[中文名][英文名][日文名][介质][分辨率][播映类型 集数/卷数][编码及封装信息][制作组名称]`

**音乐**：`艺术家名 - 专辑名 - 年代 - 格式 - 媒介 - 来源（可选）`
- 例：`张靓颖 - 第七感 - 2014 - FLAC 分轨 - CD - 幻听`

### 1.2 标题规则（阅听类）

**电子书**：`中文名 [外文名（如有）]`
- 例：`哥伦布与大航海时代 The Life and Voyages of Christopher Columbus`

**有声书**：`中文名 [外文名（如有）]-发布组名称`
- 例：`丘吉尔传：我的青春-HDSAB`

**电子书副标题**：`作者 | 出版年份 | 出版社 | 文件格式`
**有声书副标题**：`原作者 | 演播单位 | 平台来源 | 格式`

### 1.3 IMDB 链接

如有 IMDB 链接必须填写。如没有 IMDB 链接，可填写豆瓣链接。

## 二、发布页面表单字段分析

### 2.1 双模式类型系统

发布页面有两个同名的 `type` 下拉框，**只选其中之一**：

#### 影视模式（`browsecat`，data-mode='4'）— 9 个

| 值 | 显示名称 |
|----|----------|
| 401 | 电影 |
| 402 | 电视剧 |
| 403 | 综艺 |
| 404 | 纪录片 |
| 405 | 动漫 |
| 406 | 音乐视频 |
| 407 | 教学 |
| 408 | 音乐 |
| 409 | 其它 |

#### 阅听模式（`specialcat`，data-mode='8'）— 2 个

| 值 | 显示名称 |
|----|----------|
| 499 | 有声书 |
| 500 | 电子书 |

**模式切换逻辑**：选择 `browsecat` 时显示 `mode_4` 行，选择 `specialcat` 时显示 `mode_8` 行。

### 2.2 影视模式字段（data-mode=4）

#### 媒介（`medium_sel[4]`）— 5 个

| 值 | 显示名称 |
|----|----------|
| 4 | MiniBD |
| 7 | Encode |
| 10 | APE/FLAC |
| 11 | DSD |
| 12 | Other |

**极简**：仅 5 个媒介，APE/FLAC 合并为一个选项（音乐媒介），DSD 独立。

#### 编码（`codec_sel[4]`）— 3 个

| 值 | 显示名称 |
|----|----------|
| 1 | H.264 |
| 2 | H.265 |
| 5 | Other |

**极简**：仅 3 个编码选项，不区分 x264/H.264。

#### 分辨率（`standard_sel[4]`）— 4 个

| 值 | 显示名称 |
|----|----------|
| 1 | 1080P |
| 2 | 1080I |
| 3 | 2160P |
| 4 | Other |

**极简**：无 720p、SD、8K 选项，最低 1080P。

#### 制作组（`team_sel[4]`）— 5 个

| 值 | 显示名称 |
|----|----------|
| 1 | HDS |
| 2 | CHD |
| 3 | FRDS |
| 4 | CMCT |
| 5 | Other |

**极简**：仅 5 个老牌制作组。

#### 影视模式缺失字段

- 无 `source_sel[4]`（影视模式无来源字段）
- 无 `audiocodec_sel[4]`（影视模式无音频编码字段）
- 无 `processing_sel[4]`（影视模式无处理方式字段）

### 2.3 阅听模式字段（data-mode=8）

#### 语言（`source_sel[8]`）— 3 个

| 值 | 显示名称 |

## 三、关键适配器设计要点

### 3.1 双模式发布（最重要）

SoulVoice 有两套完全不同的发布模式，通过 `type` 选择切换：

| 维度 | 影视模式（mode=4） | 阅听模式（mode=8） |
|------|-------------------|-------------------|
| type 值 | 401-409 | 499-500 |
| medium_sel | 媒介（Encode/MiniBD/APE+FLAC/DSD） | （无此字段） |
| codec_sel | 视频编码（H.264/H.265/Other） | （无此字段） |
| standard_sel | 分辨率（1080P/1080I/2160P/Other） | （无此字段） |
| team_sel | 制作组（HDS/CHD/FRDS/CMCT/Other） | （无此字段） |
| source_sel | （无此字段） | 语言（中文/英文/其它） |
| audiocodec_sel | （无此字段） | 电子书格式（AZW3/EPUB/MOBI/PDF） |
| processing_sel | （无此字段） | 书籍分类（历史/小说/社科等13类） |

适配器需要根据资源类型决定使用哪个模式，并填充对应字段。

### 3.2 字段语义三重特殊

同一个表单中，同一个字段名在不同模式下含义完全不同：

| 字段名 | 影视模式含义 | 阅听模式含义 |
|--------|------------|------------|
| `source_sel` | 不存在 | **语言** |
| `audiocodec_sel` | 不存在 | **电子书格式** |
| `processing_sel` | 不存在 | **书籍分类** |

### 3.3 影视模式字段极简

影视模式仅有 4 个质量字段（medium/codec/standard/team），缺少 source_sel、audiocodec_sel、processing_sel，是所有分析站点中影视类字段最少的。

### 3.4 阅听类 H&R 标签

电子书和有声书**必须勾选 H&R 标签**。

### 3.5 阅听类标题可含中文

与多数站不同，电子书/有声书的标题**使用中文**，影视类标题使用英文。
