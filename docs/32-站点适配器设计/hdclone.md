## 一、总则与账号规则

### 1.1 账号保留规则

| 条件 | 保留规则 |

| Veteran User 及以上等级 | 永远保留 |
| Elite User 及以上等级 | 封存账号后不会被删除 |

| 无流量用户 | 连续 **10天** 不登录将被删除 |

### 1.2 封禁规则

- 注册多个 HDClone 账号将被禁止
- 第一次捣乱行为警告，第二次永久无缘 HDClone
- 一切作弊账号会被封
- 不要把本站种子文件上传到其他 Tracker

## 二、促销规则

### 2.1 随机促销

| 概率 | 促销类型 |
|------|----------|
| 10% | 50% 下载 |
| 5% | 免费(Free) |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传(2XFREE) |

### 2.2 自动促销

| 条件 | 促销类型 |
|------|----------|
| 文件总体积 **>20GB** | 自动免费 |
| Blu-ray Disk、HD DVD 原盘 | 免费 |
| 电视剧等每季的**第一集** | 免费 |
| 关注度高 | 管理员设置促销 |
| 所有种子发布 **1个月** 后 | 自动永久 **2x 上传** |

### 2.3 促销时限

- 除了"2x 上传"以外，其余类型的促销限时 **7天**（自种子发布时起）
- "2x 上传"**无时限**
- 所有种子发布 1 个月后自动永久 2x 上传

## 三、上传规则

### 3.1 上传总则

- 上传者必须对上传的文件拥有合法的传播权
- 做种时间不足 **24小时** 或故意低速上传，将被警告甚至取消上传权限
- 发布者将获得 **双倍上传量**

### 3.2 上传者资格

- 任何人都能发布资源
- 有些用户需要先在候选区提交候选
- **游戏类资源**：只有上传员及以上等级，或管理组特别指定用户，可自由上传；其他用户须候选

### 3.3 允许的资源

- 高清视频（蓝光原碟/remux/HDTV/重编码≥720p/高清DV）
- 标清视频（来源于高清媒介的标清重编码≥480p、DVDR/DVDISO、DVDRip、CNDVDRip）
- 无损音轨（FLAC、Monkey's Audio 等，含 cue 表单）
- 5.1声道或以上音轨（DTS、DTSCD 镜像等）、评论音轨
- PC 游戏（必须为**原版光盘镜像**）
- **7日内**发布的高清预告片
- 高清相关软件和文档

### 3.4 不允许的资源

- 总体积小于 **100MB**（高清软件/文档、单曲专辑除外）
- 标清 upscale 视频
- 劣质视频（CAM、TC、TS、SCR、DVDSCR、R5、R5.Line、HalfCD 等）
- RealVideo 编码（RMVB/RM）、flv 文件
- 单独的样片
- 未达到 5.1 声道的有损音频（MP3、WMA 等）
- 无正确 cue 表单的多轨音频
- 硬盘版/高压版游戏、非官方游戏镜像、第三方 mod、小游戏合集、单独破解或补丁
- RAR 等压缩文件
- 重复(dupe)资源、禁忌敏感内容、损坏文件、垃圾文件

### 3.5 Dupe 判定规则

**来源媒介优先级**：
```
Blu-ray/HD DVD > HDTV > DVD > TV
```

**详细规则**：
- 高清版本使标清版本判定为重复
- **动漫类**：HDTV 和 DVD 有**相同优先级**（特例）
- 同媒介同分辨率重编码：按发布组确定优先级
- 总会保留一个当前最佳画质来源经重编码而成的 **DVD5 大小**（4.38GB 左右）的版本
- 基于无损截图对比，高质量版本使低质量版本视为重复
- 跨区域不同配音/字幕的 Blu-ray/HD DVD 原盘不视为重复
- 无损音轨：每个资源只保留一个版本，**分轨 FLAC** 优先级最高

**Dupe 豁免**：
- 新版本没有旧版本已确认错误/画质问题，或新版本来源质量更好 → 允许发布
- 旧版本连续断种 **45日以上** 或已发布 **18个月以上** → 不受 Dupe 约束
- 新版本发布后，旧的重复版本保留至断种

### 3.6 资源打包规则（试行）

允许打包：
- 按套装售卖的高清电影合集
- 整季电视剧/综艺节目/动漫
- 同一专题纪录片
- **7日内**高清预告片
- 同一艺术家 MV（标清按 DVD 打包、高清需分辨率相同）
- 同一艺术家音乐（**5张以上**专辑方可打包；**两年内**专辑可单独发布）
- 分卷动漫剧集、角色歌、广播剧
- 发布组打包资源

打包要求：相同媒介、相同分辨率、编码一致、发布组统一

### 3.7 标题命名规则

- 电影：`[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
  - 范例：`蝙蝠侠:黑暗骑士 The Dark Knight 2008 PROPER 720p BluRay x264-SiNNERS`
- 电视剧：`[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
  - 范例：`越狱 Prison Break S04E01 PROPER 720p HDTV x264-CTU`
- 音轨：`[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]`
- 游戏：`[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]`

### 3.8 例外规则

- 允许发布 TV/DSR 体育类标清视频
- 允许发布 <100MB 高清相关软件和文档
- 允许发布 <100MB 单曲专辑
- 允许发布 **2.0声道及以上**国语/粤语音轨
- 允许附带字幕、游戏破解补丁、字体、包装扫描图（统一打包或不打包）
- 允许发布音轨时附带附赠 DVD 文件

## 四、盒子规则

- 允许使用盒子，但**禁止共享类型**盒子（共享 IP、硬盘等）
- 需到论坛客服区备案（IPV4/IPV6、时间、带宽、客户端版本、是否独立 IP）
- 请勿删除备案信息，保留购买凭证
- 盒子用户可**正常享受种子优惠**（区别于多数站的黑种规则）
- 建议使用 **100MB/s** 限速规则

## 六、字幕区规则

- 允许格式：srt/ssa/ass/cue/zip/rar
- Vobsub(idx+sub)或其他格式/合集需打包上传
- 不允许 lrc 歌词或非字幕/cue 文件
- 不合格判定：不匹配/不同步/打包错误/编码错误/语种标识错误/命名不明确/重复
- 举报奖励 **50魔力**，上传者扣除 **100魔力**

以下标准 NexusPHP 字段在 HDClone **不存在**：

## 八、关键适配器设计要点

### 8.1 字段极简

HDClone 是目前分析过的 NexusPHP 站点中字段最精简的之一：
- 无 source_sel（来源/地区）
- 无 audiocodec_sel（音频编码）
- 无 processing_sel（处理方式）
- 仅 5 个标签
- 仅 14 个制作组（无 Other）
- 无匿名发布选项

### 8.2 短剧分类

有独立的 Playlet/短剧（type=409）分类。如果源站有短剧内容，映射到此分类；否则"其他"为 type=407。

### 8.3 AV1 编码支持

编码字段支持 AV1（codec_sel=3），适配器需能识别 AV1 编码。

### 8.4 data-mode='4'

所有质量字段带 `[4]` 后缀（如 `medium_sel[4]`、`codec_sel[4]`），对应 `data-mode='4'`。
