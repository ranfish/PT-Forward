# 好学 站点适配器设计

## 站点信息

| 项目 | 内容 |
|------|------|
| 站点名称 | 好学|
| 站点地址 | https://www.hxpt.org |
| 站点框架 | NexusPHP（深度定制，教育专题） |
| Tracker URL | https://tracker.hxpt.org/announce.php |
| 特殊功能 | **纯教育类专题站**，字段含义完全不同于常规 PT 站，火花货币，价格系统(30%税率)，认领系统，签到+勋章 |
| 认证方式 | JWT 格式 `c_secure_pass`（非标准 NexusPHP md5 hash） |
| 规则页面 | rules.php |
| 发布页面 | upload.php |

## 二、完整站点规则（rules.php）

> 数据来源：rules.php（6851 字符）

### 2.1 总则

- 不允许发送垃圾信息
- 一切作弊帐号会被封
- 注册多账号将被禁止
- 不要把本站种子上传到其他 Tracker
- 第一次捣乱警告，第二次永久无缘好学

### 2.2 账号保留规则

| 等级/状态 | 保留条件 |
|-----------|----------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上（封存） | 封存后不会被删除 |
| 封存账号 | 连续 400 天不登录删除 |
| 未封存账号 | 连续 150 天不登录删除 |
| 无流量用户 | 连续 100 天不登录删除 |

### 2.3 促销规则

**随机促销概率**（种子上传后系统自动）：

| 概率 | 促销类型 |
|------|----------|
| 10% | 50% 下载 |
| 5% | 免费 |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传 |

**自动免费**：
- 文件总体积 >20GB → 自动免费
- Blu-ray Disk / HD DVD 原盘 → 自动免费
- 电视剧每季第一集 → 免费
- 关注度高的种子由管理员定夺

**促销时限**：
- 除"2x 上传"外，其余促销限时 **7 天**
- "2x 上传"无时限
- **所有种子发布 1 个月后自动永久成为 2x 上传**

### 2.4 上传总则

- 上传者须保证上传速度与做种时间（不足 24h 或故意低速将被警告/取消上传权限）
- **发布者获得双倍上传量**
- 游戏类资源只有上传员及以上等级可自由上传，其他用户须先候选

### 2.5 上传者资格

- 任何人都能发布资源
- 部分用户需先在候选区提交候选

### 2.6 允许的资源

- 高清视频：蓝光/HD DVD 原碟/remux/HDTV/重编码(≥720p)/高清 DV
- 标清视频：高清媒介重编码(≥480p)/DVDR/DVDISO/DVDRip/CNDVDRip
- 无损音轨及 cue（FLAC、APE 等）
- 5.1 声道及以上音轨（DTS、DTS CD 镜像等）、评论音轨
- PC 游戏（必须原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 2.7 不允许的资源

- 总体积 <100MB
- 标清 upscale 视频
- 枪版（CAM/TC/TS/SCR/DVDSCR/R5/R5.Line/HalfCD）
- RealVideo 编码/rmvb/rm/flv 文件
- 单独样片（须和正片一起上传）
- 未达 5.1 声道的 MP3/WMA 等有损音频
- 无正确 cue 的多轨音频
- 硬盘版/高压版游戏、非官方镜像、第三方 mod、小游戏合集、单独破解/补丁
- RAR 等压缩文件
- Dupe 资源
- 色情/敏感政治内容
- 损坏文件/垃圾文件

### 2.8 Dupe 判定规则

**来源优先级**：Blu-ray/HD DVD > HDTV > DVD > TV（高优先级使低优先级被判定为 Dupe）

**详细规则**：
- 同一视频高清版本使标清版本被判定为 Dupe
- 动漫类 HDTV 和 DVD 有相同优先级（特例）
- 相同媒介+相同分辨率：按发布组优先级判定（参考 Scene & Internal 规范）
- 保留一个 DVD5 大小（~4.38GB）的最佳画质重编码版本
- 基于无损截图对比，高质量使低质量被判定为 Dupe
- 不同区域/不同配音/字幕的 Blu-ray/HD DVD 原盘不被视为 Dupe
- 每个无损音轨原则上只保留一个版本（**分轨 FLAC 优先级最高**）
- 新版本允许发布条件：无旧版已确认错误/新来源更高质量
- **旧版连续断种 45 日或已发布 18 个月以上**，新版不受 Dupe 约束
- 新版发布后旧版保留直至断种

### 2.9 打包规则

允许打包：
- 套装高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7 日内高清预告片
- 同一艺术家 MV（标清按 DVD 打包，高清须同分辨率）
- 同一艺术家音乐（5 张以上专辑方可打包，两年内可单独发布）
- 分卷动漫/角色歌/广播剧
- 发布组打包资源

打包要求：视频须相同媒介+相同分辨率+相同编码（预告片例外），电影合集须同一发布组，音频须相同编码格式

### 2.10 标题命名规范

**电影**：`[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组`

**电视剧**：`[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组`

**音轨**：`[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组]`

**游戏**：`[中文名] 名称 [年份] [版本] [发布说明][-发布组]`

### 2.11 PT-Gen 支持来源

支持 **imdb / douban / bangumi / indienova** 四种来源链接

### 2.12 价格系统

- 发布者可设定种子价格
- 用户下载时发布者获得收入，扣除 **30% 税率**

### 2.13 认领系统

- 用户可认领种子（显示 [N/1000]）

### 2.14 货币

- 站点货币为**火花**（非魔力值）
- 签到获得 1000 火花
- 有火花商店、勋章系统、装饰品中心

## 四、HXPT 特殊注意事项

### 4.1 教育专题站

HXPT 是**纯教育类 PT 站**，所有字段含义被重新定义：

| 标准 NexusPHP 字段名 | HXPT 实际含义 |
|----------------------|---------------|
| `medium_sel` | 资源载体（视频/音频/书籍/文档/课件...） |
| `codec_sel` | 文件格式（MP4/PDF/EPUB/ZIP...） |
| `audiocodec_sel` | 适用年级（幼儿园→高三） |
| `processing_sel` | 教材版本（人教版/苏教版/部编版...） |
| `source_sel` | 资源年份（2010年前→2026） |
| `standard_sel` | 视频分辨率（保留原含义） |
| `team_sel` | 制作组（HX/HXPT/HXWEB/Other） |

### 4.2 适配器无法通用

由于字段含义完全重定义，HXPT **不能使用标准的字段映射逻辑**。适配器需要：
- 完全独立的映射表
- 根据资源类型判断填写（教育资源 vs 影视资源逻辑完全不同）
- 标签含学科分类，需要根据资源内容选择

### 4.3 Cloudflare 防护

站点使用 Cloudflare bot management，curl/Playwright 无头浏览器无法通过 TLS 指纹检测，需在有浏览器环境时采集数据。

### 4.4 JWT 认证

使用 JWT 格式的 `c_secure_pass` cookie（非标准 NexusPHP md5 hash），包含 `user_id` 和 `expires` 字段。
