
## Dupe判定规则

与织梦/自然站**完全一致**：

| 规则 | 说明 |
|------|------|
| 媒介优先级 | Blu-ray/HD DVD > HDTV > DVD > TV |
| 高清优于标清 | 高清版本使标清版本被判定为重复 |
| 动漫特例 | HDTV和DVD版本有相同优先级 |
| 同媒介同分辨率 | 按发布组优先级确定 |
| DVD5保留 | 总会保留一个DVD5大小（4.38GB）的版本 |
| 截图对比 | 基于无损截图对比 |
| 不同区域原盘 | 不同配音/字幕不算重复 |
| 无损音轨 | 原则上只保留一个版本，分轨FLAC优先级最高 |
| 断种替换 | 连续断种45日或发布18个月以上 |

## 促销规则

### 随机促销
| 概率 | 促销类型 |
|------|---------|
| 10% | 50%下载 |
| 5% | 免费 |
| 5% | 2x上传 |
| 3% | 50%下载 & 2x上传 |
| 1% | 免费 & 2x上传 |

### 自动免费条件
- 文件总体积 > 20GB → 自动免费
- Blu-ray / HD DVD 原盘 → 自动免费
- 电视剧每季第一集 → 自动免费

### 促销时限
- 除"2x上传"外，其余促销限时 **7 天**
- "2x上传"**无时限**
- 所有种子发布 **1 个月**后自动永久"2x上传"

## 允许/禁止的资源

### 允许
- 高清视频（Blu-ray/HD DVD 原碟、Remux、HDTV、高清重编码 ≥720p、高清 DV）
- 标清视频（高清媒介标清重编码 ≥480p、DVDR/DVDISO、DVDRip/CNDVDRip）
- 无损音轨及 cue 表单（FLAC/APE 等）
- 5.1 声道及以上音轨、评论音轨
- PC 游戏（原版光盘镜像）
- 7 日内高清预告片
- 高清相关软件和文档

### 禁止
- 总体积 < 100MB
- 标清 upscale 视频
- CAM/TC/TS/SCR/DVDSCR/R5/HalfCD
- RealVideo/RMVB/RM/FLV
- 单独样片
- < 5.1 声道有损音频
- 无正确 cue 多轨音频
- 硬盘版/高压版游戏
- RAR 压缩文件
- 重复/色情/敏感内容
- 损坏文件/垃圾文件
