# NicePT 站点适配器设计

> ⛔ **禁止发布 — 纯 9KG/成人内容站**

## 站点信息

| 项目 | 值 |
|------|-----|
| 站点名称 | NicePT |
| 域名 | www.nicept.net |
| 框架 | NexusPHP |
| 站点定位 | 纯 9KG/成人内容站，不适合做源站和发布站 |
| 建站时间 | 2019 年 |
| 发布页面 | `upload.php` |
| 提交地址 | `takeupload.php`（POST multipart/form-data） |
| Tracker | `https://www.nicept.net/announce.php` |

| 认领 | 1000 种 |

---

## 一、站点内容定位

仅可发布以下类型内容：
- 德艺双馨老师视频作品
- 动漫（限制级）
- 写真、套图
- 情色三级
- 直播类内容

**禁止**：幼、兽。

## 二、促销规则

### 2.1 随机促销

| 概率 | 促销类型 |
|------|---------|
| 10% | 50% 下载 |
| 5% | 免费 |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传 |

### 2.2 自动免费

- 文件总体积 **> 200GB** → 自动免费
- **Blu-ray Disk / HD DVD 原盘** → 自动免费
- 电视剧等 **每季第一集** → 自动免费
- 关注度高的种子 → 管理员手动促销

### 2.3 促销时限

- 除"2x 上传"外，其余限时 **7 天**
- "2x 上传"无时限
- **国产资源 = 无优惠**
- 不定期全站免费

## 三、H&R 规则

| 项目 | 内容 |
|------|------|
| 考察期限 | 下载完成后 **600 小时**内 |
| 做种要求 | 累计做种 **> 99 小时** 或 分享率 **> 2** |
| H&R 上限 | **20 个**，达到此值账号禁用 |
| 消除方式 | **20,000 魔力值**消除 1 个 H&R |
| 全站范围 | 所有种子均受 H&R 约束 |

> **注意**：论坛帖子 #6675 中记录的旧规则（288h/72h/200%）已过时，以 rules.php 页面为准。

## 四、发布规则

### 4.1 发布注意事项

- 鼓励单个视频发布
- 打包发布必须通过候选，否则种子会被删除
- 发布种子必须包含不低于 **3 张截图** 及 **MediaInfo** 信息
- 违规发种者，版主有权剥夺 Free/置顶等权利，严重违规直接删帖

### 4.2 标题规则

- 必须注明 **【国家】** 和 **【有无码】**
- 视频资源需标明题材（动画/动漫/游戏/套图等）
- 有番号的尽量在副标题里表明番号以及方便分类的关键词
- 副标题补充说明资源特点

### 4.3 简介规则

- 禁止直接使用其他 PT 站点的图片链接
- 禁止任何外部链接
- 正文尽可能详细描述资源内容、出处
- **强制不少于 5 张图片**（可使用封面、视频截图）

## 五、发布页面表单字段分析

### 5.1 基础信息字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| `file` | file | ✓ | 种子文件 |
| `name` | text | - | 标题 |
| `small_descr` | text | - | 副标题 |
| `nfo` | file | - | NFO 文件（Power User 以下不可查看） |
| `descr` | textarea | ✓ | 简介（BBCode） |
| `uplver` | checkbox | - | 匿名发布 |

**注意**：无 url/pt_gen/cover/screenshots/technical_info 字段。极简表单。

### 5.2 类型（`type`）— 8 个（data-mode='4'）

| 值 | 显示名称 | 分类 |
|----|----------|------|
| 500 | 日本有码 | 限制级 |
| 401 | 日本无码 | 非限制级 |
| 402 | 欧美 | 非限制级 |
| 403 | 动漫（限制级） | 限制级 |
| 404 | 写真、套图 | 非限制级 |
| 501 | 其他（限制级） | 限制级 |
| 503 | 真人秀，自拍（限制级） | 限制级 |
| 504 | SM调教（限制级） | 限制级 |

**特点**：分类以内容类型+地区+限制级划分，500 系列为限制级分类，400 系列为非限制级分类。

### 5.3 来源（`source_sel[4]`）— 6 个

| 值 | 显示名称 |
|----|----------|
| 1 | Blu-ray |
| 2 | HD DVD |
| 3 | DVD |
| 4 | HDTV |
| 5 | TV |
| 6 | Other |

### 5.4 编码（`codec_sel[4]`）— 6 个

| 值 | 显示名称 |
|----|----------|
| 1 | H.264 |
| 2 | VC-1 |
| 3 | Xvid |
| 4 | MPEG-2 |
| 5 | Other |
| 13 | HEVC |

### 5.5 分辨率（`standard_sel[4]`）— 5 个

| 值 | 显示名称 |
|----|----------|
| 11 | 720p 以下 |
| 3 | 720p |
| 2 | 1080i |
| 1 | 1080p |
| 10 | 4K 2160p |

**特点**：ID 非标准排序。720p 以下 = 11，4K = 10。无 8K。

### 5.6 标签（`tags[4][]`）— 6 个 checkbox

| 值 | 显示名称 |
|----|----------|
| 1 | 禁转 |
| 2 | 首发 |
| 4 | DIY |
| 5 | 国语 |
| 6 | 中字 |
| 7 | HDR |

### 5.7 缺失字段

以下常见 NexusPHP 字段在本站**不存在**：
- `medium_sel` — 无媒介字段
- `audiocodec_sel` — 无音频编码字段
- `team_sel` — 无制作组字段
- `url` — 无 IMDb 链接字段
- `pt_gen` — 无 PT-Gen 字段
- `cover` / `screenshots` — 无独立海报/截图字段（在简介中展示）
- `technical_info` — 无独立 MediaInfo 字段（在简介中展示）
