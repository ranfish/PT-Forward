### 允许的资源
- 写真、人像、风光、纪实、人文、儿童、超现实、动物、食物

### 禁止的资源

- **瀑布流展示**：torrents.php?display_mode=waterfall（图片站特有）
- **国家标签筛选**：按国家筛选作品

## 发布页面字段

### 表单字段

| 字段 | name | 必填 | 说明 |
|------|------|------|------|
| 种子文件 | `file` | 是 | file input |
| 类型 | `type` | 是 | 下拉选择（仅特殊区：教程/软件/图书/预设） |
| 标签 | `tags[4][]` | 否 | checkbox 多选（图片内容标签） |
| 标签 | `tags[5][]` | 否 | checkbox 多选（国家标签） |
| 标题 | `name` | 否 | |
| 简介 | `descr` | 是 | BBCode 编辑器 |
| 匿名 | `uplver` | 否 | checkbox |

### 表单提交
- action: `takeupload.php`
- 方法: POST + multipart/form-data

### 关键发现

- **上传表单无 type（分类）下拉框**：图片分类通过导航栏 cat 参数选择，上传时默认跟随当前分类
- **type 下拉仅出现于特殊区**（教程/软件/图书/预设），特殊区使用 `special.php`
- **无质量下拉框**：无 medium_sel/codec_sel 等标准 NexusPHP 质量字段
- **无 IMDb 链接字段**：图片站无影视关联
- **双标签组**：tags[4] 和 tags[5] 分别对应内容标签和国家标签
- **无副标题字段**：无 small_descr

## 质量字段

**无任何标准 NexusPHP 质量下拉框。** 没有 medium_sel、codec_sel、audiocodec_sel、team_sel、resolution_sel、source_sel、processing_sel。

### 促销规则

**随机促销**（种子上传后系统自动随机）：

| 概率 | 类型 |
|------|------|
| 10% | 50%下载 |
| 5% | 免费 |
| 5% | 2x上传 |
| 3% | 50%下载 & 2x上传 |
| 1% | 免费 & 2x上传 |

**自动促销**：
- 文件总体积 > 20GB：自动免费
- Blu-ray/HD DVD 原盘：自动免费
- 发布 1 个月后：自动永久 2x上传

**促销时限**：
- 除 2x上传外，其余限时 7 天
- 2x上传无时限

## 站点规则（来源：rules.php）

### 总则

- 不允许发送垃圾信息
- 一切作弊账号会被封
- 禁止注册多个账号
- 禁止把本站种子上传到其他 Tracker
- 第一次捣乱警告，第二次永久封禁

### 账号保留规则

| 条件 | 处理 |
|------|------|
| Veteran User 及以上 | 永远保留 |
| Elite User 及以上封存账号 | 不被删除 |
| 封存账号连续 400 天不登录 | 删除 |
| 未封存账号连续 150 天不登录 | 删除 |
| 无流量用户连续 100 天不登录 | 删除 |

### 上传总则

- 上传者必须对上传文件拥有合法传播权
- 保证上传速度与做种时间（不足24小时将警告甚至取消上传权限）
- 发布者获得双倍上传量

### 盒子规则（2026年1月15日启用）

- 单种上传速度限制：500Mbps
- 分享倍率限制：种子发布72小时内仅享受 种子体积×3 的上传量
- 需在控制面板手动登记盒子 IP
- VIP 及工作组以上等级不受限制
- 试图绕过盒子规则视为作弊，永久封禁
- 处罚：未登记第一次警告4周（有解封机会），第二次不予解封；超速同

## 特殊说明

1. **纯图片站**：PT 圈罕见的图片专题站，禁止视频/音频/软件/游戏
2. **不适合做源站和发布站**：没有其它站点与瞬间类似，无法从其它 PT 站转发图片资源
3. **地区标签体系**：以国家/地区（CN/JP/FR/UK/US/KR/GER/OTHER）为标签核心
4. **无质量字段**：图片资源不涉及视频编码/音频编码/分辨率等传统质量维度
5. **双标签组**：tags[4] 有 20 个标签，tags[5] 有 15 个标签（子集）
6. **盒子规则完善**：500Mbps 限速 + 72h 分享倍率限制
7. **瀑布流展示**：特有的 waterfall 展示模式，适合图片浏览
8. **特殊区独立入口**：教程/软件/图书/预设通过 special.php 访问
9. **候选制**：Crazy User 以下需候选，Crazy User 及以上可直接发布
10. **暂不接受捐赠**
11. **无做种积分要求**：升级条件仅看注册时间+下载量+分享率
