#### 使用示例

##### 基础使用
```javascript
const { chromium } = require('playwright');

const browser = await chromium.launch({
  headless: true,
  args: ['--no-sandbox', '--disable-setuid-sandbox', '--ignore-certificate-errors']
});

const page = await browser.newPage();
await page.goto('https://example.com');
const title = await page.title();

await browser.close();
```

##### Cookie 认证
```javascript
await page.context().addCookies([
  { name: 'session', value: 'xyz', domain: 'example.com', path: '/' }
]);
await page.goto('https://example.com');
```

##### JavaScript 执行
```javascript
const data = await page.evaluate(() => {
  return {
    title: document.title,
    content: document.body.innerText
  };
});
```

##### 页面截图
```javascript
await page.screenshot({ 
  path: '/tmp/screenshot.png', 
  fullPage: false 
});
```

#### 推荐配置

##### PT 站点访问配置
```javascript
const browser = await chromium.launch({
  headless: true,
  args: [
    '--no-sandbox',
    '--disable-setuid-sandbox',
    '--ignore-certificate-errors',
    '--disable-web-security',
    '--disable-features=VizDisplayCompositor'
  ]
});

const context = await browser.newContext({
  viewport: { width: 1920, height: 1080 },
  userAgent: 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
});

const page = await context.newPage();
```

##### 性能优化
```javascript
// 并发控制
const browser = await chromium.launch();
const pages = await Promise.all([
  browser.newPage(),
  browser.newPage(),
  browser.newPage()
]);

// 资源控制
const context = await browser.newContext({
  // 限制并发连接
  maxConcurrentConnections: 6,
  // 超时设置
  navigationTimeout: 30000,

  acceptDownloads: false

```

#### 安全说明

##### 安全使用原则
- ❌ **绝不保存** Cookie、passkey、apikey、token、令牌等敏感信息
- ❌ **绝不写入** 本地文件、日志、数据库、版本控制
- ❌ **绝不上传** 云端存储、外部服务
- ✅ **仅内存使用** - 本次会话临时变量
- ✅ **会话结束** - 自动清除所有内存数据

##### Cookie 使用方式

**方案 A：用户临时提供 Cookie（推荐）**
```
用户提供 Cookie → 本次会话内存使用 → 会话结束自动清除
```

**方案 B：用户自行运行**
```
我提供测试脚本 → 用户自行运行 → 返回结果给我
```

**方案 C：分段分析**
```
用户分次提供规范内容 → 我分析规则 → 不需要完整 Cookie
```
