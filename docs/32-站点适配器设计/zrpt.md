## 标题命名规范

```
[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称

### 纪录片（剧集类）

[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称

### 音轨
```

## 简介规范

### 必须包含
1. **海报/封面**（必须）
2. **演职员名单 + 剧情概要**
3. **文件的详细情况**：格式、时长、编码、码率、分辨率、语言、字幕
4. **画面截图**（尽量包含）

## Dupe判定规则

与织梦站规则高度一致：

| 规则 | 说明 |
|------|------|
| 媒介优先级 | Blu-ray/HD DVD > HDTV > DVD > TV |
| 高清优于标清 | 高清版本使标清版本被判定为重复 |
| 动漫特例 | HDTV和DVD版本有相同优先级 |
| 同媒介同分辨率 | 按发布组优先级确定 |
| DVD5保留 | 总会保留一个DVD5大小（4.38GB）的版本 |
| 截图对比 | 基于无损截图对比，高质量版本使低质量版本被视重复 |
| 不同区域原盘 | 不同配音/字幕的blu-ray/HD DVD不算重复 |
| 无损音轨 | 原则上只保留一个版本，分轨FLAC优先级最高 |
| 断种替换 | 旧版本连续断种45日或发布18个月以上，新版本不受约束 |

## 资源打包规则

与织梦站规则高度一致：

- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- **同一专题纪录片**
- 7日内高清预告片
- 同一艺术家MV（标清按DVD打包；分辨率相同的HD MV）
- 同一艺术家音乐（5张以上方可打包，两年内可单独发布）
- 分卷动漫剧集/角色歌/广播剧
- 发布组打包

## 字幕区规则

### 允许格式
srt / ssa / ass / cue / zip / rar

- Vobsub格式(idx+sub)或其他格式须打包为zip/rar
- 开放音轨对应cue表单上传，多个cue须打包
- 禁止lrc歌词或非字幕/cue文件

### 不合格判定
- 与种子不匹配 / 与视频不同步 / 打包错误
- 包含无关文件或垃圾信息 / 编码错误 / cue错误 / 语种标识错误
- 标题命名不明确或含冗余信息

### 字幕奖惩
- 举报不合格字幕：确认后奖励50魔力（3天内发放）
- 上传不合格字幕：扣100魔力
- 恶意上传/举报：额外扣魔力甚至警告

## 注意事项

1. **仅支持发布纪录片**：分类为纪录片题材，非标准影视分类
2. **所有字段ID与织梦站完全不同**：媒介/分辨率/音频编码/编码的ID映射需使用本文档实际数据，不能套用织梦站
3. **全站Free**：2025-09-04~2026-06-04期间全站免费
4. **规则与织梦高度同模板**：Dupe判定、打包规则、标题格式与织梦站几乎完全一致
5. **有编码字段(codec_sel)**：织梦站没有独立的编码下拉框，但本站有（9个选项含AV1/VP9）
6. **technical_info字段**：独立MediaInfo textarea，织梦站无此字段
7. **制作组偏少**：仅7个（HDS/CHD/MySiLU/WiKi/Other/ZRPT）
