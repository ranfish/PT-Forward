## 一、发布页面表单字段分析

### 1.1 基础信息字段

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| `file` | file | ✓ | 种子文件 |
| `name` | text | - | 标题 |
| `small_descr` | text | - | 副标题 |
| `url` | text | - | IMDb 链接（`data-pt-gen="url"`） |
| `nfo` | file | - | NFO 文件 |
| `descr` | textarea | ✓ | 简介（BBCode） |
| `uplver` | checkbox | - | 匿名发布（value="yes"） |

### 1.2 PT-Gen 集成

字段 `data-pt-gen` 出现在 `url` 和 `pt_gen` 两个属性中。外部信息链接支持 **五来源**：
- **IMDb**（`url` 字段）
- **Douban**（豆瓣）
- **Bangumi**（番组计划）
- **Indienova**（独立游戏平台）
- **PT-Gen**（自动填充）

### 1.3 缺失字段

- **`source_sel`**：无来源字段
- **`processing_sel`**：无处理方式字段
- **`dburl`**：无独立豆瓣字段（豆瓣通过 PT-Gen 间接支持）

## 二、站点完整规则

> 来源：`rules.php`

### 2.1 总则

- 不要做管理员明文禁止的事情。
- 不允许发送垃圾信息。
- 一切作弊的账号会被封。
- 注册多个 LemonHD 账号的用户将被禁止。
- 不要把本站的种子文件上传到其他 Tracker。
- 第一次在论坛或服务器中的捣乱行为会受到警告，第二次永久封禁。

### 2.2 账号保留规则

| 条件 | 规则 |
|------|------|
| Ultimate User 及以上 | 永远保留 |
| Veteran User 及以上封存账号 | 不会被删除 |
| 封存账号用户 | 连续 **100 天**不登录 → 封禁 |
| 未封存账号用户 | 连续 **50 天**不登录 → 封禁 |
| 没有流量用户（上传/下载都为 0） | 连续 **14 天**不登录 → 封禁 |
| 被封禁账号 | 连续 **180 天**不登录 → 从数据库彻底物理删除 |

- 单种上传速度限制在 **100MB/s**，全局上传速度不做限制。

### 2.3 上传总则

- 上传者必须对上传的文件拥有合法的传播权。
- 上传者必须保证上传速度与做种时间。**如果在其他人完成前撤种或做种时间不足 24 小时，或者故意低速上传**，上传者将会被警告甚至取消上传权限。
- 对于自己发布的种子，**发布者将获得 5 倍的上传量**。
- 如果有一些违规但却有价值的资源，可告知管理组，可能破例允许发布。

### 2.4 上传者资格

- **任何人都能发布资源。**
- 有些用户需要先在候选区提交候选。
- **游戏类资源**：只有上传员及以上等级，或管理组特别指定的用户才能自由上传。其他用户必须先在候选区提交候选。

### 2.5 允许的资源

- 高清（HD）视频：蓝光原碟、HD DVD 原碟、Remux、HDTV 流媒体、高清重编码（≥720p）、高清 DV 等
- 标清（SD）视频：只能来源于高清媒介的标清重编码（≥480p）、DVDR/DVDISO、DVDRip、CNDVDRip
- 无损音轨（及相应 CUE 表单）：FLAC、Monkey's Audio 等
- 5.1 声道或以上标准的电影/音乐音轨（DTS、DTS-CD 镜像等）、评论音轨
- PC 游戏（必须为原版光盘镜像）
- 7 日内发布的高清预告片
- 与高清相关的软件和文档

### 2.6 禁止的资源

- 总体积小于 **100MB** 的资源
- 标清 upscale 或部分 upscale 而成的视频
- 劣质标清视频：CAM、TC、TS、SCR、DVDSCR、R5、R5.Line、HalfCD 等
- RealVideo 编码（RMVB/RM）、flv 文件
- 单独的样片（须与正片一起上传）
- 未达到 5.1 声道标准的有损音频文件（如 MP3、有损 WMA）
- 无正确 CUE 表单的多轨音频文件
- 硬盘版/高压版游戏、非官方游戏镜像、第三方 mod、小游戏合集、单独破解或补丁
- RAR 等压缩文件
- 重复（Dupe）资源
- 禁忌/敏感内容
- 损坏的文件
- 垃圾文件（病毒、木马、网站链接、广告文档等）

### 2.7 重复（Dupe）判定规则

**来源媒介优先级**：Blu-ray/HD DVD > HDTV > DVD > TV。同一视频高优先级版本将使低优先级版本被判定为重复。

**其他规则**：
- 同一视频的高清版本将使标清版本被判定为重复。
- **动漫类特例**：HDTV 版本和 DVD 版本有相同优先级。
- 相同媒介、相同分辨率的高清重编码：按发布组确定优先级，高优先级发布组版本将使低/同级发布组版本被判定为重复。
- **总会保留一个**当前最佳画质的来源经重编码而成的 DVD5 大小（约 4.38 GB）的版本。
- 基于无损截图对比，高质量版本将使低质量版本被视为重复。
- 来自其他区域、包含不同配音和/或字幕的 Blu-ray/HD DVD 原盘版本**不被视为重复**。
- 每个无损音轨资源原则上只保留一个版本，**分轨 FLAC 格式有最高优先级**。
- 对站内已有资源：
  - 新版本没有旧版本已确认错误/画质问题，或新版本来源有更好质量 → 允许发布，旧版本被视为重复
  - 旧版本已连续断种 **45 天**以上或已发布 **18 个月**以上 → 发布新版本不受 Dupe 约束
- 新版本发布后，旧重复版本将被保留直至断种。

### 2.8 资源打包规则

**允许打包的资源**：
- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7 日内高清预告片
- 同一艺术家的 MV（标清 MV 只允许按 DVD 打包，不允许单曲 MV 单独发布；高清 MV 须分辨率相同）
- 同一艺术家的音乐（5 张或以上专辑方可打包；两年内发售的专辑可单独发布；打包时应剔除站内已有资源或都包括进来）
- 分卷发售的动漫剧集、角色歌、广播剧等
- 发布组打包发布的资源

**打包要求**：视频必须来源媒介相同、分辨率相同、编码格式一致（预告片例外），电影合集发布组也须统一。音频必须编码格式一致（如全部为分轨 FLAC）。打包发布后，将视情况删除相应单独种子。

### 2.9 例外条款

- 允许发布来源于 TV/DSR 的体育类标清视频。
- 允许发布 <100MB 的高清相关软件和文档。
- 允许发布 <100MB 的单曲专辑。
- 允许发布 2.0 声道或以上的国语/粤语音轨。
- 允许附带字幕、游戏破解与补丁、字体、包装扫描图（须统一打包或统一不打包）。
- 允许发布音轨时附带附赠 DVD 相关文件。

### 2.10 种子信息规范

**标题命名**：
- 电影：`[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 电视剧：`[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 音轨：`[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]`
- 游戏：`[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]`

**外部信息**：电影和电视剧必须包含外部信息链接（如 IMDb 链接）。

**简介要求**：
- 电影/电视剧/动漫：必须含海报/横幅/封面；尽可能含截图、详细文件信息、演职员及剧情概要。
- 体育节目：不得在文字介绍/截图/文件名/文件大小/时长中泄露比赛结果。
- 音乐：必须含专辑封面和曲目列表。
- PC 游戏：必须含海报/封面；尽可能含截图。

### 2.11 促销规则

**随机促销**（种子上传后自动随机）：

| 概率 | 促销类型 |
|------|----------|
| 10% | 50% 下载 |
| 5% | 免费 |
| 5% | 2x 上传 |
| 3% | 50% 下载 & 2x 上传 |
| 1% | 免费 & 2x 上传 |

**自动促销**：
- 文件总体积 > 20GB → 自动免费
- Blu-ray Disk / HD DVD 原盘 → 自动免费
- 电视剧等每季第一集 → 自动免费
- 所有种子发布 **1 个月**后自动永久成为 2x 上传

**促销时限**：除"2x 上传"无时限外，其余促销限时 **7 天**（自发布时起）。

## 三、关键适配器设计要点

### 3.1 双语分类名

分类名使用中英双语（如 `Movies(电影)`），适配器需匹配中英文部分。

### 3.2 4K/8K 双重体现

4K/8K 同时出现在媒介（medium_sel）和分辨率（standard_sel）中，发布时需同步填写：
- 媒介选 4K-UltraHD（6）或 8K-UltraHD（7）
- 分辨率选 4K_UHD（1）

### 3.3 极简编码和制作组

编码仅 5 种，制作组仅 6 个，适配器映射简单。但源站有更多编码/制作组时，可能大量映射到 "Other"。

### 3.4 PT-Gen 多来源

支持 IMDb + Douban + Bangumi + Indienova + PT-Gen 五种来源，适配器可充分利用自动填充。

### 3.5 3D 独立分类

有 3D 视频独立分类（408），适配器需识别 3D 内容。

### 3.6 Dupe 规则

标准 NexusPHP dupe 规则：Blu-ray/HD DVD > HDTV > DVD > TV。动漫类 HDTV 和 DVD 同优先级。

### 3.7 发布者 5 倍上传量

发布者获得 5 倍上传量奖励（非标准的 2 倍），站内发布积极性较高。
