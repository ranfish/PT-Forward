## 发种规范（来源：wiki.ptskit.org/zh/torrent）

### 标题规范
格式：**剧名 季数 年份 分辨率 来源媒介 视频编码 音频编码-制作组**

示例：
- `Infinity S04 2024 2160p WEB-DL H.265 AAC-ADWeb`
- `Friends S01 1994 1080p WEB-DL H265 AAC2.0-CSWEB`
- `Bossy.CEO.Falls.In.Love 2024 1080p WEB-DL H265 AAC2.0-PTSKIT`

### 必要信息
- **分辨率**：720p / 1080p / 2160p / 4K
- **视频编码**：H.264 / H.265 / HEVC / x264 / x265
- **音频编码**：AAC / AC3 / DTS / DDP / FLAC / Atmos
- **标题必须纯英文**，不允许中文字符，不允许 `<>|?*` 非法字符

### 副标题（简介）
- 中文名称 + 集数信息 + 类型标签 + 主创信息（可选）
- 示例：`老友记 第一季 / Friends Season 1 / 全24集 / 喜剧 / 詹妮弗·安妮斯顿 主演`

### MediaInfo
- **必须提供**完整 MediaInfo 技术信息
- **最低 500 字符**
- 包含视频参数、音频参数、字幕信息

### 图片要求
| 类型 | 数量 | 说明 |
|------|------|------|
| 海报 | ≥ 1 张 | 必须 |
| 截图 | ≥ 3 张或 1 张九宫格 | 必须 |

### 标签选择
- **官方种子**：站点制作组原创，无需审核自动通过
- **转载种子**：必须标注"转载"标签 + 类别标签（电影/电视剧/动漫等），需审核
- **转载种子不标注"转载"标签将被自动拒绝**

### 转载规则
**禁止：**
- 单集/分集种子（必须完整季度包）
- 同一用户同一内容的多个 WEB 画质
- 已有高画质版本后再发低画质

**允许：**
- 完整季度包
- 更高画质版本
- 非 WEB 来源（BluRay、HDTV、DVDRip）
- 不同用户可发相同内容不同画质

## 站点规则（来源：rules.php）

### 总则

- 不允许发送垃圾信息
- 一切作弊账号会被封
- 禁止注册多个账号
- 禁止把本站种子文件上传到其他 Tracker
- 第一次捣乱警告，第二次永久封禁

### 账号保留规则

| 条件 | 处理 |
|------|------|
| Extreme User 及以上 | 永远保留 |
| Elite User 及以上封存账号 | 不被删除 |
| 封存账号连续 400 天不登录 | 删除 |
| 未封存账号连续 30 天不登录 | 删除 |
| 无流量用户连续 7 天不登录 | 删除 |

### 促销规则

**随机促销**（种子上传后系统自动随机）：

| 概率 | 类型 |
|------|------|
| 10% | 50%下载 |
| 5% | 免费 |
| 5% | 2x上传 |
| 3% | 50%下载 & 2x上传 |
| 1% | 免费 & 2x上传 |

**自动促销**：
- 文件总体积 > 20GB：自动免费
- Blu-ray/HD DVD 原盘：自动免费
- 电视剧每季第一集：自动免费
- 发布 1 个月后：自动永久 2x上传

**促销时限**：
- 除 2x上传外，其余促销限时 7 天
- 2x上传无时限

### 上传总则

- 上传者必须对上传文件拥有合法传播权
- 保证上传速度与做种时间（不足24小时将警告甚至取消上传权限）
- 发布者获得双倍上传量
- 游戏类资源：仅上传员及以上等级可自由上传，其他用户需候选

### 允许的资源

- 高清视频：蓝光原碟/HD DVD/remux/HDTV/720p+重编码/高清DV
- 标清视频：仅限来源于高清媒介的 480p+重编码、DVDR/DVDISO/DVDRip
- 无损音轨（FLAC/Monkey's Audio 等）+ cue 表单
- 5.1声道及以上音轨
- PC 游戏（必须原版光盘镜像）
- 7日内高清预告片
- 高清相关软件和文档

### 禁止的资源

- 总体积 < 100MB
- 标清 upscale 视频
- CAM/TC/TS/SCR/DVDSCR/R5/HalfCD 等低质量
- RealVideo/RMVB/RM/FLV
- 单独样片
- < 5.1 声道的有损音频（MP3/WMA）
- 无正确 cue 的多轨音频
- 硬盘版/高压版游戏、非官方镜像、第三方 mod、小游戏合集
- RAR 等压缩文件
- 重复资源
- 色情/敏感政治内容
- 损坏文件/垃圾文件

### Dupe 判定规则

- 来源优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 高清版本使标清版本被判定为重复
- 动漫类 HDTV 和 DVD 同优先级（特例）
- 同媒介同分辨率：按发布组优先级判定
- 总会保留一个 DVD5 大小（4.38GB）的版本
- 无损截图对比：高质量使低质量被判定为重复
- 不同区域配音/字幕的蓝光原盘不视为重复
- 无损音轨原则上只保留一个版本（分轨 FLAC 优先级最高）
- 断种 45 日或发布 18 个月以上：不受 dupe 约束

### 资源打包规则（试行）

允许打包：
- 按套装售卖的高清电影合集
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 7日内高清预告片
- 同一艺术家 MV（标清按 DVD 打包，高清需分辨率相同）
- 同一艺术家音乐（5张以上专辑，两年内可单独发布）
- 分卷发售动漫剧集/角色歌/广播剧
- 发布组打包资源

打包要求：视频必须同媒介/同分辨率/同编码，音频必须同编码格式。

### 新手考核

- 上传增量：30 GB
- 做种积分增量：2500

### 邀请规则

- 邮箱限制：@qq.com / @foxmail.com / @gmail.com
- 上家连 ban：3 名受邀者未完成新手考核取消邀请权限；10 名取消下载权限

### 认领规则

- 发布 15 天后可认领
- 每种子最多 10 人认领，每人最多认领 2000 个
- 达标标准：每月做种≥360h 或上传≥10 倍体积
- 达标种子魔力值 2 倍奖励

### 盒子规则

- 需报备登记
- 下载速度超 100 MB/s 且未登记将禁用下载权限

### 字幕区规则

- 允许格式：srt/ssa/ass/cue/zip/rar
- 不合格字幕直接删除，上传者扣 100 魔力值
- 举报不合格字幕奖励 50 魔力值

## 特殊说明

1. **仅源站**：只支持官种，不支持普通用户发布，不适合作为发布目标站
2. **主打短剧**：短剧/动态漫/擦边剧三大主打分类禁止转入转出
3. **开放分类可转**：电影/电视剧/动漫支持转入转出，可作为这三个分类的源站

5. **多域名多 Tracker**：4 个域名 + 3 个 Tracker 地址
6. **H&R 规则**：有做种时间要求，下载后需保种
7. **制作组 PTSKIT**：站组后缀 PTSKIT、CSWEB、ADWeb 等
8. **tag_id 分类系统**：使用 tag_id 而非标准 cat 参数进行分类导航
9. **标题纯英文要求**：不允许中文字符和非法字符
10. **转载标签必填**：转载种子必须标注"转载"标签，否则自动拒绝
11. **禁止单集发布**：必须发布完整季度包
12. **种子审核系统**：转载种子有审核流程（未审/通过/拒绝）
