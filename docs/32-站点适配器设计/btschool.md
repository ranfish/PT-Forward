## 站点特色

- **模板A组基准站**：与阳光/壹吧/樱花构成模板A组，上传/Dupe/打包/促销规则完全一致，其他三站均引用学校站为模板来源
- **教育特色**：独立资料/Education分类（ID=402），计算机学习/专业资料/英语资料命名规则
- **游戏类限制**：仅上传员及以上等级可直接发布，其他用户必须先候选
- **极简字段**：无source_sel/processing_sel/地区字段，无tags[]标签系统
- **标签用span[]**：使用`span[]`而非`tags[]`，仅6个标签
- **无制作组来源**：制作组12个，含跨站组（WiKi/HDChina/HDB_iNT/M-team/CMCT/Ourbits）
- **分辨率合并**：1080p/i合并为一项，720p/i合并为一项
- **音频编码合并**：DTS-HD/DTS合并为一项
- **媒介含DVDRip**：无独立UHDTV，无Track/MiniBD
- **促销自动**：新种随机促销（6档概率），>20GB自动Free，蓝光/HD DVD原盘自动Free，电视剧每季首集自动Free，1个月后永久2X
- **上传限速**：25MB/s，超过100MB/s自动封号
- **发布者双倍上传量**

## 标题命名规范

### 通用规则
- **主标题必须全部英文或拼音**，中文名一律放到副标题
- 副标题中文名用`[]`括起来，多个中文名用`/`分隔
- 副标题自定义标签用`**`在最后括起来

### 电影类
- **主标题**：英文名 年代 介质 分辨率 编码 (音频)-制作组
- **副标题**：[中文名1/中文名2]
- **示例**：`Avatar 2010 BluRay 720p x264 DTS-TorrentGUI` / `[阿凡达/天神下凡]`

### 剧集类
- **主标题**：英文名 第几季第几集 介质 分辨率 编码 (音频)-制作组
- **副标题**：[中文名]（[集数]）[剧集类型]
- **示例**：`CSI LV S01E01 HDTV 720p x264-CTU` / `[犯罪现场调查][第1季第1集][美剧]`

### 动漫类
- **Rip**：英文名或罗马音 集数 分辨率 类别 编码 (音频)-压片组(字幕组）
- **ISO/BDMV**：英文名或罗马音 集数 类别 区号
- 动漫是唯一使用BDRip/HDTVRip的分类

### 综艺类
- **主标题**：英文名或中文拼音 (集数或完整日期) 介质 分辨率 编码 (音频)-制作组
- **副标题**：[中文名][完整日期或集数]（[电视台]）

### 音乐类
- **主标题**：艺术家英文名 专辑英文名 版本年份 音频格式-(制作者信息)
- **副标题**：[艺术家中文或英文名][专辑中文或英文名][音频格式]

### 游戏类
- **主标题**：平台 游戏类型 英文名字 格式-小组信息
- **副标题**：[中文名][游戏类型]（[游戏格式]）

### 纪录片类
- **主标题**：(播放电视台) 英文名 （第几季第几集） 介质 分辨率 编码-(制作组)
- **副标题**：[中文名]（[集数]）[电视台或频道]

## 资源打包规则（试行）

允许打包的资源类型：
1. 按套装售卖的高清电影合集（如 The Ultimate Matrix Collection Blu-ray Box）
2. 整季的电视剧/综艺节目/动漫
3. 同一专题的纪录片
4. 7日内的高清预告片
5. 同一艺术家的MV（标清MV只允许按DVD打包，不允许单曲MV单独发布；高清MV分辨率须相同）
6. 同一艺术家的音乐（5张或5张以上专辑方可打包；两年内发售的专辑可单独发布）
7. 分卷发售的动漫剧集、角色歌、广播剧等
8. 发布组打包发布的资源

打包约束：
- 视频资源：相同媒介类型、相同分辨率、相同编码格式（预告片例外），电影合集还须发布组统一
- 音频资源：编码格式一致（如全部为分轨FLAC）
- 打包时应剔除站内已有的资源，或将它们都包括进来

## 字幕区规则

- 允许格式：srt / ssa / ass / cue / zip / rar
- Vobsub格式（idx+sub）或其他格式须打包为 zip/rar
- 禁止上传 lrc 歌词等非字幕/cue 文件
- 不合格字幕（不匹配/不同步/打包错误/编码错误/重复等）直接删除，上传者扣100魔力
- 举报不合格字幕奖励50魔力

## 发布规则

### 禁止内容
- 总体积小于100MB的资源
- 标清upscale视频
- CAM/TC/TS/SCR/DVDSCR/R5/HalfCD
- RMVB/RM/flv文件
- 有损MP3/有损WMA（未达5.1声道标准）
- RAR等压缩文件
- 硬盘版/高压版游戏
- 色情（含三级）资源

### Dupe规则
- 来源媒介优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 动漫类HDTV与DVD同优先级（特例）
- 断种45日或发布18个月以上可重发
- 官方组发布后同版本视为Dupe

### H&R规则
- 下载完成后**10天内**做种**20小时**（种子不分大小，时间一样）
- 上传量 ≥ 下载量 → 直接通过，无需保种20小时
- HR未达标数 ≥ 10 → 封禁（VIP及以上不降级）
- 免除HR：20000魔力/种
- HR封禁解封：捐赠100元，HR置为0
- H&R显示格式：`H&R: 考察中数(黑色)/未达标数(红色)`
  - 考察中：需保种的种子，做种时间未完成且上传量<下载量
  - 已达标：完成做种时间或上传量≥下载量
  - 未达标：未完成做种时间且上传量<下载量
  - 已免罪：大赦（特殊情况）

### 认领规则
- 所有会员无论等级均可认领
- 每种最多20人认领，每人最多100种
- 月做种≥300小时 或 月上传量≥种子体积10倍
- 系统每月1日凌晨统计上月数据并清零
- 种子体积>1GB且发布>30天才有魔力奖励
- 未达标自动回收认领资格（不扣魔力）
- 已达标数据清零，继续保留认领资格
- 主动放弃认领扣300魔力
- 奖励标准：
  - 常规资源（非官方组）：500魔力/月
  - 官方资源（官方组发布）：1000魔力/月

### 限速规则
- 上传限速25MB/s
- 超过100MB/s系统自动封号（封号解封：上传量减半）
- 非发布者上传速度超25MB/s → **3倍扣罚上传量**
  - 示例：汇报数据 96.26 GB @ 28.42 MB/s → 扣减 96.26×3=288.78 GB
- 超速2次以上：警告小红帽
- 超速5次以上：上传量减半，累次再减半，严重封号
- 发布者须在个人说明中标明盒子信息或宽带情况

## 转载注意事项

1. **极简字段**：无source_sel/processing_sel/地区，无独立的IMDb链接字段（仅填编号）
2. **标签用span[]**：字段名是`span[]`而非`tags[]`，仅6个标签，无HDR/DV标签
3. **分辨率合并**：1080p/i合并，720p/i合并，无法区分逐行/隔行
4. **音频合并**：DTS-HD/DTS合并为一项，无DDP/EAC3/Atmos
5. **制作组含跨站组**：转载其他站官种时，可能能匹配到WiKi/HDChina/M-team等跨站组
6. **禁止选官组**：非管理组禁止选择BTSCHOOL/Zone/BtsHD/BtsTV/BtsPAD
7. **标题严格英文**：主标题必须是英文/拼音/数字，不能含中文
8. **游戏类需候选**：非上传员用户发布游戏必须先候选
9. **IMDb/豆瓣仅填编号**：不同于其他站填URL，学校只填编号部分
10. **无PT-Gen**：无PT-Gen一键获取简介功能
