## 三、上传规则

### 3.1 上传总则

1. 上传者必须对上传的文件拥有合法的传播权
2. **严禁发布国家版权局重点作品版权保护预警名单**所包含的作品，违规直接禁用账号
3. **本站不允许发布 R18 内容**
4. 做种时间不足 **48小时** 或故意低速上传，将被警告、强制候选甚至取消上传权限、封禁账号
5. **非本站小组成员的作品，禁止使用本站官组名称(PTCafe)作为标识和后缀**
6. 转载其他 PT 站点种子**禁止删除原站后缀、修改文件名**等操作
7. 官方保种组不得对本站种子进行限速（临时性不低于 100KB/S，长期性不低于 1MB/S）

### 3.2 上传者资格

- 任何人都能发布资源
- 咖啡新人及以下等级用户需要先在**候选区**提交候选
- 提交候选后请耐心等候审核，不要发送站内信催促

### 3.3 上传奖励

| 行为 | 奖励 |
|------|------|
| 发布一个新种子 | **50 咖啡豆** |
| 上传一个新字幕 | **25 咖啡豆** |
| 发布者上传量 | 该种子实际大小的 **1.5倍** |

### 3.4 不允许的资源

- 电子书、音乐、游戏、学习资料等非影视资源
- 劣质视频文件（CAM、TC、TS 等）
- 纯外文，无中文信息资源
- 版权风险较大的资源
- 其他管理员认为不适合发布的资源

### 3.5 已完结电视剧规则

- 已完结电视剧，请直接发布**合集**，并勾选【完结】标签，不得拆分发布
- 已经发布的分集资源，请及时跟进到完结，不得烂尾
- 转载的资源，请保持原目录结构发布，以便于自动化辅种

### 3.6 转载规则

1. 确认资源本身不是禁转（违规转载友站禁转标签种子，第一次警告一个月，第二次直接封禁）
2. **原样转载**：不得对原种子文件（夹）重新命名；不得擅自篡改原种子相关信息（截图张冠李戴、文件与 NFO 不一致等）；不得搞"大杂烩"
3. 无法申明来源及其他疑似转自公网的资源不予通过
4. 转载违规处罚：**删种起步、上不封顶**

## 四、超速与限速规则

### 4.1 超速规则

- 单种上传速度超过 **100MB/S**（发布员及 VIP 亦遵循）
- 第一次：禁用上传下载权限 **24小时**
- 第二次：禁用上传下载权限 **7天**
- 第三次：**禁用账号**

### 4.2 恶意限速

- 判定：单人做种 **6小时内**上传速度稳定在 **100Kb/s** 以下
- 处罚：查实恶意限速，封禁账号

## 六、认领与求种规则

### 6.1 认领规则

**认领条件**：
- 所有用户均可认领种子
- 种子发布 **7天** 后可以认领
- 一个种子最多被 **10个** 用户认领

**达标要求**（每月）：
- 月做种时间 **≥500小时**，或月上传量 **≥种子大小的10倍**
- 允许放弃认领，每次每种子需支付 **1000 咖啡豆**

**结算规则**：
- 系统每月 1 日统计上月认领情况
- 首月达标按奖励标准发咖啡豆（**达标种子魔力奖励是正常魔力值的 2倍**）
- 首月未达标则该月不计奖励，认领记录保持
- 非首月不达标：删除认领并扣除 **3000 咖啡豆**

### 6.2 求种规则

- 求种者须提供 **TMDB**（国外资源必须）链接或豆瓣链接
- 10集以下剧集：保种 **10天** 以上可撤种
- 10-50集剧集：保种 **15天** 以上可撤种
- 50集以上剧集：保种 **1个月** 以上才能撤种
- 第1次违规：禁用下载上传权限1周；第2次：直接禁用账号

## 七、官种命名规范

> 来源：论坛 topicid=77 + topicid=973

### 7.1 主标题格式

```
影片名称 [S**] 年份 分辨率 [来源平台] 媒介 音频编码 [画质技术] [帧率] 视频编码-制作组
```

范例：`The Glory S01 2023 2160p NF WEB-DL DDP5.1 Atmos DV 60fps H265-PTCafe`

| 字段 | 说明 | 必填 |
|------|------|------|
| 影片名称 | 强制使用英文名，无英文名则用拼音（首字母大写） | ✓ |
| S** | 季号，电影不填 | 剧集必填 |
| 年份 | 上映年份 | ✓ |
| 分辨率 | 如 2160p、1080p | ✓ |
| 来源平台 | 如 Netflix、Amazon，国内平台不填 | 选填 |
| 媒介 | 如 WEB-DL、Bluray | ✓ |
| 音频编码 | 如 DDP5.1 Atmos、AAC | ✓ |
| 画质技术 | DV/HDR/SDR，仅 4K 及以上填写 | 选填 |
| 帧率 | 仅 ≥60fps 时填写 | 选填 |
| 视频编码 | H264/AVC 统一填 H264，HEVC/H265 统一填 H265 | ✓ |
| 制作组 | PTCafe/CafeTV/CafeWEB/Diy@PTCafe | ✓ |

### 7.2 副标题格式

```
中文名 集数 | 分辨率 画质技术 帧率 | 类型:XX | 主演:XX [其他信息]
```

范例：`黑暗荣耀 全16集 | 4K 杜比视界 120帧 | 类型:剧情 | 主演:宋慧乔/李到晛/林智妍/廉惠兰/朴成焄 【双语简繁字幕】`

| 字段 | 说明 |
|------|------|
| 中文名 | 影片中文名及其他译名 |
| 集数 | 已完结填"全XX集"，未完结填"第XX集"（个位数补0，如"第01集"） |
| 分辨率 | 仅 2160p 以上填写，2160p 填"4K" |
| 画质技术 | DV 填"杜比视界"，其余填英文 |
| 帧率 | 仅 >60fps 时填写 |
| 类型 | 影片类型信息 |
| 主演 | 演员信息 |
| 其他信息 | 中括号内填写（是否硬字幕、有无水印等） |

### 7.3 文件名与文件夹名

- **文件名**：按主标题命名，用"点"替代空格
  - 范例：`3.Body.Problem.S01E01.2024.2160p.Netflix.WEB-DL.H265.DV.60fps.DDP5.1.Atmos-PTCafe.mkv`
- **文件夹名**：
  - 电影：与文件同名
  - 电视剧：去掉集数
  - 大陆平台资源文件夹名前需添加中文，国外资源不允许添加中文
  - 范例：`三体.3.Body.Problem.S01.2024.2160p.Netflix.WEB-DL.H265.DV.60fps.DDP5.1.Atmos-PTCafe`

## 十、关键适配器设计要点

### 10.1 source_sel 含义特殊

PTCafe 的 `source_sel` 表示 **地区**（大陆/港台/欧美/日本/韩国/印度/其他），而非其他站常见的 **来源**（Blu-ray/HDTV/WEB-DL 等）。适配器需注意字段语义差异。

### 10.2 data-mode='4' 字段后缀

所有质量字段带 `[4]` 后缀（如 `source_sel[4]`、`medium_sel[4]`），对应 `data-mode='4'`。提交时需使用带后缀的字段名。

### 10.3 已完结剧集要求合集

已完结电视剧禁止分集发布，必须发布合集并勾选【完结】标签。

### 10.4 分辨率合并

1080p 和 1080i 合并为 `standard_sel=3`（1080p/1080i/FHD），适配器无需区分这两种分辨率。

### 10.5 MediaInfo/BDInfo 独立字段

有 `technical_info` 独立字段用于 MediaInfo/BDInfo，而非嵌入简介中。
