## 站点特色

- **原盘特效站**：以原盘资源为主，适合做源站下载资源
- **不适合做发布站**：用户等级为User时无直接发布权限，需走候选区
- **规则与学校站高度相似**：上传规则、Dupe规则、打包规则、促销规则几乎完全一致
- **游戏类限制**：与学校站相同，仅上传员及以上可直接发布
- **促销自动**：随机促销 + >20GB自动Free + 蓝光原盘自动Free + 1个月后永久2X
- **发布者双倍上传量**

## 标题命名规范（来自rules.php）

### 电影类
- `[中文名] 名称 [年份] [剪辑版本] [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 例：`蝙蝠侠:黑暗骑士 The Dark Knight 2008 PROPER 720p BluRay x264-SiNNERS`

### 剧集类
- `[中文名] 名称 [年份] S**E** [发布说明] 分辨率 来源 [音频/]视频编码-发布组名称`
- 例：`越狱 Prison Break S04E01 PROPER 720p HDTV x264-CTU`

### 音轨类
- `[中文艺术家名 - 中文专辑名] 艺术家名 - 专辑名 [年份] [版本] [发布说明] 音频编码[-发布组名称]`
- 例：`恩雅 - 冬季降临 Enya - And Winter Came 2008 FLAC`

### 游戏类
- `[中文名] 名称 [年份] [版本] [发布说明][-发布组名称]`
- 例：`红色警戒3:起义时刻 Command And Conquer Red Alert 3 Uprising-RELOADED`

## 发布规则

### 禁止内容
- 总体积小于100MB的资源
- 标清upscale视频
- CAM/TC/TS/SCR/DVDSCR/R5/HalfCD
- RMVB/RM/flv文件
- 有损MP3/有损WMA（未达5.1声道标准）
- RAR等压缩文件
- 硬盘版/高压版游戏
- 色情/敏感政治内容

### Dupe规则
- 来源媒介优先级：Blu-ray/HD DVD > HDTV > DVD > TV
- 动漫类HDTV与DVD同优先级（特例）
- 断种45日或发布18个月以上可重发

### 账号保留
- Extreme User及以上：永久保留
- Elite User及以上封存后：不删除
- 封存账号：400天未登录删除
- 未封存账号：150天未登录删除
- 无流量用户：14天后封禁

## 转载注意事项

1. **仅源站角色**：本站不适合做发布站，仅适合作为源站下载资源
2. **无发布权限**：User等级无直接发布权限，无法获取完整表单结构
3. **规则与学校站高度相似**：几乎完全相同的上传/Dupe/打包规则，可能是同一套模板
4. **需升级后才能分析**：需达到发布权限等级后才能获取完整upload.php表单数据
5. **标题格式含中文前缀**：标题格式为`[中文名] 英文名...`，中文名在方括号中放在最前面
