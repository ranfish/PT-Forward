## 三、发种规则（rules.php）

### 3.1 上传总则

- 上传者必须对上传的文件拥有合法的传播权
- 做种时间不足24小时或故意低速上传将被警告甚至取消上传权限
- **非本站小组成员禁止使用本站官组名称作为标识和后缀**
- 发布一个新种子得到50憨豆奖励，上传字幕得20憨豆
- 发布者获得该种子正常实际大小的上传量（非双倍）

### 3.2 上传者资格

- 任何人都能发布资源
- 发布信息多次填写错误/不规范可能被警告、强制候选甚至取消上传权限

### 3.3 允许的资源

- 高清视频（Blu-ray/Remux/重编码≥720p）
- 标清视频（来源于高清的≥480p重编码）
- 7日内高清预告片
- 高清相关软件和文档
- 体育标清视频（TV/DSR）
- <100MB 高清软件/文档/单曲专辑

### 3.4 不允许的资源

- 标清 upscale 视频
- 被管理员判定的有素材制作疑问的资源

### 3.5 Dupe 规则

- **从他站转载官组 WEB 资源时，搜索本站官组是否已发布**
- 同质量、同分辨率的 WEB 资源会被判定为 Dupe
- **官组发布后的资源会把之前同质量、同分辨率资源删除**（但保留不同作品）
- **禁止转载 HDR+DV 种子**，请单独发布 HDR 或 DV 版本

### 3.6 打包规则

- 高清电影合集（套装）
- 整季电视剧/综艺/动漫
- 同一专题纪录片
- 打包视频须：相同媒介、相同分辨率、相同编码
- **禁止发布无意义非发行方打包 DIY/Remux 合集**
- **禁止一个 IMDB/豆瓣号下正片多分辨率打包**（除非审核组允许）

### 3.7 保种组规则

- 保种组成员不得对种子限速（临时≥100KB/s，长期≥1MB/s）
- 收到管理组 PM 需24小时内回应
- 累计3次未回应将被降级

### 3.8 种子促销

- 新种子默认 free 8h（不区分是否官种）
- 大体积种子可用憨豆购买促销：免费1天(7,000)/置顶1天(14,000)/置顶+免费1天(27,000)

## 三B、Wiki 上传规则（补充）

> 数据来源：Wiki 上传规则页面

### 3B.1 禁止发布的资源

- **禁止发布未完结番剧/动画的分集资源**（须等全集完结后发布整季/全集打包）
- **辅种须对种子做完整校验**，禁止不校验直接做种

### 3B.2 制作组黑名单

**禁止转载以下制作组的资源**：

| 黑名单制作组 | 说明 |
|-------------|------|
| FGT | 禁止 |
| Hao4K | 禁止 |
| Mp4Ba | 禁止 |
| RARBG | 禁止 |

### 3B.3 标题命名规范

- 标题须使用 **0day 命名**（英文标准命名格式）
- 从他站转载时标题须**原样保留**，禁止擅自修改

### 3B.4 转载规则

- 转载资源须原样保留原标题、原简介
- 转载官组 WEB 资源时，须先搜索本站是否已有同质量同分辨率的官种发布
- **禁止转载 HDR+DV 合并种子**，须分别发布 HDR 和 DV 版本

### 3B.5 特殊禁止项

- **禁止 HDR+DV 合并发种**：转载时遇到 HDR+DV 合并的种子，须拆分为 HDR 和 DV 分别发布
- **禁止 AI 插帧/补帧资源**
- **音乐类仅限官组发布**
- **高清视频须≥1080p**（低于 1080p 的高清资源不允许发布）

## 三C、盒子及限速规则（Wiki）

> 数据来源：Wiki 盒子及限速规则页面

### 3C.1 盒子下载计量

- 盒子下载量 **100% 计量**（即盒子下载视为黑种，上传者不获得上传量奖励）

### 3C.2 上传限速规则

- **72 小时内上传量不得超过种子体积的 3 倍**

## 四、HHanClub 特殊注意事项

### 4.1 全站官种

站点全站官种，**只能做源站，不能做发布站**。在 PT-Forward 中应标记为 `PublishEnabled=false`、`SourceEnabled=true`。

### 4.2 Checkbox 筛选系统

种子列表使用 checkbox 多选筛选（非下拉框），字段名带 `[]` 后缀。这使得 RSS URL 构造和页面解析与标准 NexusPHP 不同。

### 4.3 processing 地区分类

`processing[]` 提供剧集地区分类（美剧/日剧/韩剧/港剧/台剧/大陆剧/英剧），是已采集站点中地区分类最细的之一。

### 4.4 独立封面/截图上传

发布页面有 `cover` 和 `screenshot` 独立文件上传字段，标准 NexusPHP 无此字段。

### 4.5 禁止 HDR+DV 合并发种

规则明确禁止转载 HDR+DV 种子，需单独发布 HDR 或 DV 版本。源站消费时需注意拆分。

### 4.6 TAA 音频编码

`audiocodec[]` 含 TAA(18) 和 ALAC(19)，较罕见。

### 4.7 憨豆货币

站点使用"憨豆"货币系统，非魔力值。

### 4.8 盒子下载全计量

盒子下载量 100% 计量（黑种），72h 内上传限 3 倍，超速 100MB/s 封禁。

### 4.9 云服务商封禁

阻断阿里云/腾讯云/华为云/百度云/京东云/UCloud 六大云服务商 IP。

### 4.10 H&R 规则

20 天内做种 72h，累计 5 次警告封禁，消除警告需 500,000 憨豆。

### 4.11 制作组黑名单（源站过滤）

转载时需过滤：FGT、Hao4K、Mp4Ba、RARBG。源站消费时若标题含这些制作组名，需标记或跳过。

### 4.12 禁止未完结分集

禁止发布未完结番剧/动画的分集资源，须等全集后整季打包。源站消费分集番剧时需注意。

### 4.13 标题 0day 命名

标题须使用 0day 英文标准命名格式，转载须原样保留。源站标题可直接使用无需转换。

### 4.14 AI 插帧禁止

禁止 AI 插帧/补帧资源。

### 4.15 音乐仅限官组

音乐类仅官组可发布，非官组音乐资源不做源站消费。

## 六、适配器实现要点

### 6.1 仅源站模式

```go
adapter.PublishEnabled = false
adapter.SourceEnabled = true
adapter.OfficialOnly = true
```

### 6.2 Checkbox 筛选 URL 构造

RSS/列表请求需使用 checkbox 参数格式：

```
torrents.php?cat[]=401&medium[]=10&standard[]=1
```

### 6.3 地区信息提取

```go
func mapProcessing(val int) string {
    switch val {
    case 3: return "US"
    case 4: return "JP"
    case 5: return "KR"
    case 6: return "HK"
    case 7: return "TW"
    case 8: return "CN"
    case 9: return "GB"
    default: return ""
    }
}
```
